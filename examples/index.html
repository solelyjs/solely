Solely 框架组件示例

使用 Solely Web Components 框架构建的组件示例。

计数器组件

一个带有增加、减少与重置功能的简单计数器。

用户卡片组件

展示用户信息，可展开详情的卡片组件。

待办列表组件

一个完整的待办事项列表，支持筛选与管理功能。

如何使用这些组件

在项目中使用 Solely 组件的步骤如下：

1. 导入组件

      <script type="module">
      // 在示例项目中，建议直接引入入口注册文件
      import './index.ts';
      // 或者按需引入各组件：
      import './components/counter/counter.ts';
      import './components/todo-list/todo-list.ts';
      import './components/user-card/user-card.ts';
      </script>

2. 在 HTML 中使用组件

      <!-- 计数器组件 -->
      <solely-counter step="2" count="3"></solely-counter>
      
      <!-- 用户卡片（通过属性传入 ID） -->
      <solely-user-card user-id="2"></solely-user-card>

如果需要传递非字符串数据或复杂对象，可以使用 JS 设置属性或 $data：

      <!-- 用户卡片通过 JS 设置 -->
      <solely-user-card id="card-js"></solely-user-card>
      
      <script>
      const card = document.getElementById('card-js');
      
      // 设置单个属性（驼峰形式）
      card.userId = 2;
      
      // 设置复杂数据
      card.$data = {
        name: 'Alice',
        status: 'online'
      };
      
      // 或者更新部分数据
      card.$data = {
        status: 'offline'
      };
      </script>

3. 属性与类型说明

  • HTML 属性值默认都是字符串，Solely 会尝试根据 $data 默认值或 props type 自动转换类型。
  • 如果需要数字、布尔或对象类型，推荐在 props 中声明 type，或者使用 JavaScript 设置 $data。

4. 组件特性

  • 计数器： 可自定义步进值，支持增加、减少、重置
  • 用户卡片： 支持动态加载用户数据、展开详情、状态指示
  • 待办列表： 支持添加、删除、标记完成、筛选与批量操作
