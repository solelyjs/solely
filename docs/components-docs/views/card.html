Card 卡片

通用卡片容器，支持标题、封面、操作区等多种布局。

基础用法

最简单的卡片用法，包含标题和内容区。

卡片内容区域，可以放置任意内容。

 
<solely-card title="卡片标题">
  <p>卡片内容区域，可以放置任意内容。</p>
</solely-card>

无边框

通过 bordered 属性控制是否显示边框，默认无边框显示阴影。

默认状态，带有阴影效果。

使用 bordered 属性显示边框。

 
<!-- 默认：无边框，有阴影 -->
<solely-card title="无边框卡片">
  <p>默认状态，带有阴影效果。</p>
</solely-card>

<!-- 带边框 -->
<solely-card title="带边框卡片" bordered>
  <p>使用 bordered 属性显示边框。</p>
</solely-card>

宽度适配

卡片默认宽度为 100%，会自动适配父容器宽度。可以通过 style 或 class 自定义宽度。

卡片宽度 = 父容器宽度（300px）

通过 style 设置 200px

通过 CSS 类控制宽度

 
<!-- 默认：宽度 100%，适配父容器 -->
<solely-card title="默认宽度">
  <p>卡片宽度 = 父容器宽度</p>
</solely-card>

<!-- 通过 style 设置固定宽度 -->
<solely-card title="自定义宽度" style="width: 200px;">
  <p>通过 style 设置 200px</p>
</solely-card>

<!-- 通过 CSS 类控制 -->
<solely-card title="自适应父元素" class="custom-width-card">
  <p>通过 CSS 类控制宽度</p>
</solely-card>

/* CSS */
.custom-width-card {
  width: 400px;
  max-width: 100%;
}

悬停效果

使用 hoverable 属性开启悬停时的上浮效果。

鼠标悬停时会有上浮阴影效果。

 
<solely-card title="可悬停卡片" hoverable>
  <p>鼠标悬停时会有上浮阴影效果。</p>
</solely-card>

小尺寸

使用 size="small" 属性设置紧凑的卡片样式。

默认尺寸的卡片内容。

小尺寸的卡片内容，更加紧凑。

 
<solely-card title="默认尺寸">
  <p>默认尺寸的卡片内容。</p>
</solely-card>

<solely-card title="小尺寸卡片" size="small">
  <p>小尺寸的卡片内容，更加紧凑。</p>
</solely-card>

带封面

使用 cover 属性设置卡片封面图片。

卡片可以包含封面图片，适合展示图片内容。

 
<solely-card
  title="带封面的卡片"
  cover="https://example.com/image.jpg">
  <p>卡片可以包含封面图片，适合展示图片内容。</p>
</solely-card>

额外操作区

使用 extra 属性或 extra 插槽在标题右侧添加操作按钮。

更多

点击"更多"按钮查看效果。

 编辑

使用 extra 插槽可以添加按钮等复杂内容。

 
<!-- 带点击事件 -->
<solely-card title="卡片标题">
  <span slot="extra">
    <solely-button type="link" size="small" @click="handleClick()">更多</solely-button>
  </span>
  <p>点击"更多"按钮查看效果。</p>
</solely-card>

<!-- 复杂内容 -->
<solely-card title="卡片标题">
  <span slot="extra">
    <solely-button type="link" size="small">编辑</solely-button>
  </span>
  <p>使用 extra 插槽可以添加按钮等复杂内容。</p>
</solely-card>

底部操作区

使用 actions 属性或 actions 插槽添加底部操作按钮。actions 属性使用 JSON 数组字符串格式，点击会触发 action 事件。

使用 actions 属性添加操作，点击可触发事件。

使用 actions 插槽可以添加按钮等复杂操作。

确认 取消 
<!-- actions 属性：JSON 数组格式，点击触发 action 事件 -->
<solely-card title="卡片标题" actions='["编辑", "删除"]' @action="handleAction(event)">
  <p>使用 actions 属性添加操作，点击可触发事件。</p>
</solely-card>

<!-- actions 插槽：自定义复杂操作 -->
<solely-card title="卡片标题">
  <p>使用 actions 插槽可以添加按钮等复杂操作。</p>
  <div slot="actions">
    <solely-button type="primary" size="small">确认</solely-button>
    <solely-button size="small">取消</solely-button>
  </div>
</solely-card>

加载状态

使用 loading 属性显示加载状态，可通过 setLoading 方法动态控制。

点击按钮切换加载状态。

 切换加载状态 
<!-- 模板 -->
<solely-card ref="loadingCard" title="动态加载示例" :loading="$data.loading">
  <p>点击按钮切换加载状态。</p>
</solely-card>
<solely-button @click="toggleLoading()">切换加载状态</solely-button>

<!-- 或通过方法控制 -->
const card = document.querySelector('solely-card');
card.setLoading(true);

API

属性

属性 说明 类型 默认值
title 卡片标题 string ''
size 卡片尺寸 'default' | 'small' 'default'
bordered 是否显示边框 boolean false
loading 是否加载中 boolean false
hoverable 是否可悬停（悬停时有上浮效果） boolean false
cover 封面图片 URL string ''
actions 底部操作区内容（JSON 数组字符串，如 '["编辑", "删除"]'） string ''
extra 右上角额外操作区内容 string ''

插槽

插槽名 说明
默认插槽 卡片内容区域
extra 标题右侧的额外操作区
actions 底部操作区

事件

事件名 说明 回调参数
action 点击 actions 属性中的操作项时触发 CustomEvent<{ index: number, action: string }>

方法

方法名 说明 参数 返回值
setLoading(loading) 设置加载状态 loading: boolean void
setTitle(title) 设置卡片标题 title: string void

类型定义

// 卡片尺寸
type CardSize = 'default' | 'small';

// 卡片属性接口
interface CardProps {
  title?: string;       // 卡片标题
  size?: CardSize;      // 卡片尺寸
  bordered?: boolean;   // 是否显示边框
  loading?: boolean;    // 是否加载中
  hoverable?: boolean;  // 是否可悬停
  cover?: string;      // 封面图片 URL
  actions?: string;      // 底部操作区内容（JSON 数组字符串）
  extra?: string;      // 右上角额外操作区
}
