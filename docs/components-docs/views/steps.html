Steps 步骤条

引导用户按照流程完成任务的分步导航条。

基础用法

最简单的用法，通过 current 属性设置当前步骤，items 属性设置步骤数据。

<solely-steps
  current="0"
  items='[{"title":"步骤一"},{"title":"步骤二"},{"title":"步骤三"}]'>
</solely-steps>

带描述

通过 description 属性为每个步骤添加描述信息。

<solely-steps
  current="1"
  items='[
  {"title":"填写信息","description":"请填写基本信息"},
  {"title":"验证身份","description":"进行身份验证"},
  {"title":"完成","description":"注册成功"}
]'>
</solely-steps>

垂直方向

设置 direction="vertical" 使用垂直方向的步骤条。

<solely-steps
  direction="vertical"
  current="1"
  items="...">
</solely-steps>

小尺寸

设置 size="small" 使用小尺寸步骤条。

<solely-steps
  size="small"
  current="1"
  items="...">
</solely-steps>

大尺寸

设置 size="large" 使用大尺寸步骤条。

<solely-steps
  size="large"
  current="1"
  items="...">
</solely-steps>

点状步骤条

设置 progress-dot 属性使用点状步骤条，适合简洁展示。

<solely-steps
  progress-dot
  current="1"
  items="...">
</solely-steps>

自定义图标

通过 icon 属性为步骤自定义图标。

<solely-steps
  current="1"
  items='[
  {"title":"登录","icon":"👤"},
  {"title":"支付","icon":"💳"},
  {"title":"完成","icon":"✓"}
]'>
</solely-steps>

自定义颜色

通过 color 属性为步骤自定义颜色，支持预设颜色（blue、green、red、orange、gray）或自定义十六进制颜色。

<solely-steps
  current="1"
  items='[
  {"title":"蓝色","color":"blue"},
  {"title":"绿色","color":"green"},
  {"title":"橙色","color":"orange"},
  {"title":"红色","color":"red"}
]'>
</solely-steps>

<!-- 支持自定义颜色值 -->
<solely-steps current="0"
  items='[
  {"title":"自定义","color":"#722ed1"}
]'>
</solely-steps>

错误状态

在步骤数据中设置 status="error" 显示错误状态。

<solely-steps
  current="2"
  items='[
  {"title":"已完成","status":"finish"},
  {"title":"出错啦","status":"error"},
  {"title":"等待中","status":"wait"}
]'>
</solely-steps>

禁用状态

设置 disabled: true 禁用特定步骤的点击。

<solely-steps
  current="0"
  items='[
  {"title":"当前步骤"},
  {"title":"已禁用","disabled":true},
  {"title":"未解锁","disabled":true}
]'>
</solely-steps>

步骤切换

使用 next() 和 prev() 方法控制步骤切换。

上一步 下一步
当前步骤: {{$data.currentStep + 1}} / 4
<!-- HTML -->
<solely-steps ref="mySteps" current="0" items="..."></solely-steps>
<solely-button @click="prev()">上一步</solely-button>
<solely-button @click="next()">下一步</solely-button>

<!-- JavaScript -->
const steps = document.querySelector('solely-steps');

steps.next();
steps.prev();
steps.setCurrent(2);

API

属性

属性名 说明 类型 默认值
current 当前步骤，从 0 开始 number 0
direction 步骤条方向，可选值为 horizontal、vertical string horizontal
size 尺寸，可选值为 default、small、large string default
items 步骤数据（JSON字符串） string []
progressDot 是否使用点状步骤条 boolean false

事件

事件名 说明 回调参数
change 点击步骤时触发 CustomEvent<{ index: number; item: StepItem }>

方法

方法名 说明 参数
setCurrent(current) 设置当前步骤 current: number
getCurrent() 获取当前步骤 -
next() 下一步 -
prev() 上一步 -
setItems(items) 设置步骤项 items: StepItem[]

类型定义

// 步骤项类型
interface StepItem {
  title: string;
  description?: string;
  icon?: string;
  status?: StepStatus;
  disabled?: boolean;
  color?: string;
}

// 步骤状态
type StepStatus = 'wait' | 'process' | 'finish' | 'error';

// 步骤条方向
type StepsDirection = 'horizontal' | 'vertical';

// 步骤条尺寸
type StepsSize = 'default' | 'small' | 'large';

插槽

插槽名 说明 插槽元素属性
default 自定义步骤内容（当使用插槽时，items 属性将被忽略） data-step: 步骤序号（从1开始）
data-title: 步骤标题
data-description: 步骤描述
disabled: 禁用该步骤
