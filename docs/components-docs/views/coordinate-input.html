CoordinateInput 经纬度输入框

用于输入经纬度坐标，支持度分秒（DMS）格式，可自动转换为十进制。

基础用法

同时输入纬度和经度，使用度分秒格式。输入完成后会自动跳转到下一个输入框。

<solely-coordinate-input></solely-coordinate-input>

仅纬度或经度

通过 type 属性设置只输入纬度或经度。

<!-- 仅纬度 -->
<solely-coordinate-input type="latitude"></solely-coordinate-input>

<!-- 仅经度 -->
<solely-coordinate-input type="longitude"></solely-coordinate-input>

不同尺寸

支持 small、medium、large 三种尺寸。

<solely-coordinate-input size="small"></solely-coordinate-input>
<solely-coordinate-input size="medium"></solely-coordinate-input>
<solely-coordinate-input size="large"></solely-coordinate-input>

禁用状态

设置 disabled 属性禁用输入。

<solely-coordinate-input disabled></solely-coordinate-input>

只读状态

设置 readonly 属性使输入框只读。

<solely-coordinate-input readonly></solely-coordinate-input>

设置默认值

通过 latitude 和 longitude 属性设置默认值（十进制）。

<!-- 北京天安门坐标 -->
<solely-coordinate-input 
    latitude="39.9042" 
    longitude="116.4074">
</solely-coordinate-input>

显示十进制值

添加 show-decimal 属性显示对应的十进制坐标值（默认不显示）。

<solely-coordinate-input show-decimal></solely-coordinate-input>

显示方向选择

添加 show-direction 属性显示 N/S/E/W 方向选择（默认不显示）。

<!-- 添加 show-direction 属性显示方向选择 -->
<solely-coordinate-input show-direction></solely-coordinate-input>

自动跳转

添加 auto-tab 属性开启输入完成后自动跳转到下一个输入框的功能（默认关闭）。

<!-- 设置 auto-tab，开启自动跳转 -->
<solely-coordinate-input auto-tab></solely-coordinate-input>

<!-- 不设置 auto-tab，关闭自动跳转 -->
<solely-coordinate-input></solely-coordinate-input>

组合使用

可以同时开启多个功能，如显示方向、显示十进制值、自动跳转等。

<solely-coordinate-input 
    show-direction
    show-decimal
    auto-tab
    latitude="39.9042"
    longitude="116.4074">
</solely-coordinate-input>

秒输入精度控制

通过 precision 属性控制秒输入的小数位数（默认3位）。输入时会根据小数位数决定是否自动跳转到下一个输入框。

precision="0"（整数秒）
precision="2"（2位小数）
precision="3"（3位小数，默认） 
<!-- 整数秒，输入12后自动跳转 -->
<solely-coordinate-input precision="0"></solely-coordinate-input>

<!-- 2位小数，输入12.34后自动跳转 -->
<solely-coordinate-input precision="2"></solely-coordinate-input>

<!-- 3位小数（默认），输入12.345后自动跳转 -->
<solely-coordinate-input precision="3"></solely-coordinate-input>

块级显示

添加 block 属性可以让组件宽度占满父容器。

<!-- 块级显示，宽度占满父容器 -->
<solely-coordinate-input block type="latitude"></solely-coordinate-input>

<solely-coordinate-input 
    block
    show-direction
    show-decimal>
</solely-coordinate-input>

事件监听

监听 input、change、focus、blur、keydown、clear 等事件来响应经纬度输入的交互。

事件日志（最近5条）：

<!-- 监听所有事件 -->
<solely-coordinate-input
  @input="handleInput(event)"
  @change="handleChange(event)"
  @focus="handleFocus(event)"
  @blur="handleBlur(event)"
  @keydown="handleKeyDown(event)"
  @clear="handleClear(event)"
></solely-coordinate-input>

// 原生事件：通过 event.target.value 获取值
function handleInput(event: Event) {
  const target = event.target as HTMLInputElement;
  console.log('input:', target.value); // { latitude, longitude }
}

function handleChange(event: Event) {
  const target = event.target as HTMLInputElement;
  console.log('change:', target.value); // { latitude, longitude }
}

// 原生事件：focus、blur
function handleFocus(event: FocusEvent) {
  console.log('focus');
}

function handleBlur(event: FocusEvent) {
  console.log('blur');
}

function handleKeyDown(event: KeyboardEvent) {
  if (event.key === 'Enter') {
    const target = event.target as HTMLInputElement;
    console.log('enter:', target.value); // { latitude, longitude }
  }
}

function handleClear(event: Event) {
  console.log('clear');
}

// 通过属性获取/设置值
const coordinateInput = document.querySelector('solely-coordinate-input');

// 获取值
const value = coordinateInput.value; // { latitude: 39.9042, longitude: 116.4074 }

// 设置值（对象形式）
coordinateInput.value = { latitude: 39.9042, longitude: 116.4074 };

// 设置值（数组形式）
coordinateInput.value = [39.9042, 116.4074];

// 使用方法
coordinateInput.getValue(); // 获取十进制坐标
coordinateInput.setValue(39.9042, 116.4074); // 设置坐标
coordinateInput.clear(); // 清空

API

属性

属性名 说明 类型 默认值
type 坐标类型：latitude（纬度）、longitude（经度）、both（两者） string 'both'
latitude 纬度值（十进制） number -
longitude 经度值（十进制） number -
value 坐标值对象（支持属性和方法操作） { latitude: number, longitude: number } | [number, number] -
disabled 是否禁用 boolean false
readonly 是否只读 boolean false
size 尺寸大小 'small' | 'medium' | 'large' 'medium'
showDirection 是否显示方向选择（N/S/E/W），添加属性即显示 boolean false
showDecimal 是否显示十进制值，添加属性即显示 boolean false
precision 秒输入精度（小数位数），控制秒的小数位数和自动跳转行为 number 3
autoTab 是否自动跳转到下一个输入框，添加属性即开启 boolean false
block 是否块级显示，设置为 true 时宽度占满父容器 boolean false

方法

方法名 说明 参数 返回值
getValue() 获取坐标值（十进制） - { latitude: number, longitude: number }
setValue(latitude, longitude) 设置坐标值（十进制） latitude: number, longitude: number void
getDmsCoordinateValue() 获取 DMS 格式的坐标值 - { latitude: DMS, longitude: DMS }
clear() 清空坐标值 - void

事件

事件名 说明 回调参数
input 输入值变化时触发（原生事件） Event
change 方向选择变化或调用 setValue 时触发（原生事件） Event
focus 输入框获得焦点时触发（原生事件） FocusEvent
blur 输入框失去焦点时触发（原生事件） FocusEvent
keydown 按下键盘按键时触发（原生事件），回车键可通过 event.key === 'Enter' 判断 KeyboardEvent
clear 调用 clear 方法时触发 Event

方法

方法名 说明 参数 返回值
getValue() 获取坐标值（十进制） - { latitude, longitude }
getDmsCoordinateValue() 获取 DMS 格式的坐标值 - { latitude: DMS, longitude: DMS }
setValue(latitude, longitude) 设置坐标值（十进制） latitude: number, longitude: number -
clear() 清空所有值 - -

类型定义

interface DMS {
  degrees: number;    // 度 (0-90 纬度, 0-180 经度)
  minutes: number;    // 分 (0-59)
  seconds: number;    // 秒 (0-59.999)
  direction: 'N' | 'S' | 'E' | 'W';  // 方向
}

interface Coordinate {
  latitude: number;   // 纬度（十进制）
  longitude: number;  // 经度（十进制）
}

interface DMSCoordinate {
  latitude: DMS;      // 纬度 DMS 格式
  longitude: DMS;     // 经度 DMS 格式
}

interface CoordinateInputProps {
  type?: 'latitude' | 'longitude' | 'both';  // 坐标类型
  latitude?: number;   // 纬度值
  longitude?: number;  // 经度值
  disabled?: boolean;  // 是否禁用
  readonly?: boolean;  // 是否只读
  size?: 'small' | 'medium' | 'large';  // 尺寸
  showDirection?: boolean;  // 是否显示方向选择
  showDecimal?: boolean;    // 是否显示十进制值
  autoTab?: boolean;        // 是否自动跳转
  precision?: number;       // 精度（小数位数）
  block?: boolean;         // 是否块级显示
}
