Slider 滑块

滑动输入条，用于在范围内选择数值。

基础用法

最简单的用法，通过 value 属性设置初始值。

<solely-slider value="30"></solely-slider>

范围与步长

通过 min、max 设置范围，step 设置步长。

<solely-slider min="0" max="100" step="10" value="50"></solely-slider>
<solely-slider min="-50" max="50" step="5" value="0"></solely-slider>

禁用状态

通过 disabled 属性禁用滑块。

<solely-slider value="30" disabled></solely-slider>

垂直模式

设置 vertical 属性可使滑块垂直显示。

<solely-slider vertical value="30"></solely-slider>

刻度标记

设置 marks 属性显示刻度标记，marks-data 传入 JSON 格式的标记数据。

<solely-slider marks marksData='{"0":"0","25":"25","50":"50","75":"75","100":"100"}' value="50"></solely-slider>

Block 属性

设置 block 属性可使滑块宽度撑满父容器。

<solely-slider block value="40"></solely-slider>

事件处理

监听 change 事件获取滑块值变化，input 事件用于实时更新。

<!-- HTML -->
<solely-slider @input="handleInput(event)" @change="handleChange(event)"></solely-slider>

<!-- JavaScript - 原生事件监听 -->
const sliderEl = document.querySelector('solely-slider');

// input 事件 - 实时更新
sliderEl.addEventListener('input', (e) => {
  const value = e.target.value;
  console.log('实时值:', value);
});

// change 事件 - 值改变时触发
sliderEl.addEventListener('change', (e) => {
  const value = e.target.value;
  console.log('最终值:', value);
});

API

属性

属性名 说明 类型 默认值
value 当前值 number min
min 最小值 number 0
max 最大值 number 100
step 步长 number 1
disabled 是否禁用 boolean false
tooltipVisible 是否显示 Tooltip boolean true
marks 是否显示刻度标记 boolean false
marksData 刻度标记数据（JSON字符串） string -
vertical 是否垂直模式 boolean false
block 是否撑满父容器宽度 boolean false

事件

事件名 说明 回调参数
change 值改变时触发 Event
input 拖动时实时触发 InputEvent

方法

方法名 说明 参数 返回值
setValue(value) 设置滑块值 value: number void
getValue() 获取当前值 - number

类型定义

// 组件属性接口
interface SliderProps {
  value?: number;           // 当前值
  min?: number;            // 最小值
  max?: number;            // 最大值
  step?: number;           // 步长
  disabled?: boolean;     // 是否禁用
  tooltipVisible?: boolean; // 是否显示 Tooltip
  marks?: boolean;        // 是否显示刻度标记
  marks-data?: string;     // 刻度标记数据（JSON字符串）
  vertical?: boolean;     // 是否垂直模式
  block?: boolean;        // 是否撑满父容器宽度
}
