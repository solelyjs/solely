Icon 图标

语义化的矢量图形，支持 SVG Sprite、自定义图标和第三方图标库，提供灵活的尺寸、颜色和旋转控制。

SVG Sprite 方式（推荐）

直接使用 SVG Sprite 文件，图标按需加载，无需打包。

// 方式一：在入口文件全局设置（推荐）
import { setIconSpritePath } from 'solely';

// 设置 SVG Sprite 路径（相对于 public 目录）
setIconSpritePath('/icons/bootstrap-icons.svg');

// 然后直接使用
<solely-icon name="house-fill"></solely-icon>
<solely-icon name="person-fill"></solely-icon>

// 方式二：每个图标单独指定 sprite
<solely-icon name="gear-fill" sprite="/icons/bootstrap-icons.svg"></solely-icon>

手动注册图标

使用 registerIcon 或 registerIcons 手动注册自定义图标。

import { registerIcon, registerIcons } from 'solely';

// 注册单个图标（六边形）
registerIcon('my-icon', '<polygon points="8,1 14.1,4.5 14.1,11.5 8,15 1.9,11.5 1.9,4.5" fill="currentColor"/>');

// 批量注册多个图标
registerIcons({
  'logo': '<circle cx="8" cy="8" r="7" fill="none" stroke="currentColor" stroke-width="1.5"/><path d="M8 4v8M4 8h8" stroke="currentColor" stroke-width="1.5" stroke-linecap="round"/>',
  'arrow': '<path d="M4 12l4-4 4 4" fill="none" stroke="currentColor" stroke-width="1.5" stroke-linecap="round" stroke-linejoin="round"/>'
});

图标尺寸

支持 small、medium（默认）、large 三种预设尺寸，或自定义数值。

<solely-icon name="star-fill" size="small"></solely-icon>
<solely-icon name="star-fill" size="medium"></solely-icon>
<solely-icon name="star-fill" size="large"></solely-icon>
<solely-icon name="star-fill" size="32"></solely-icon>
<solely-icon name="star-fill" size="48"></solely-icon>

图标颜色

使用 color 属性自定义图标颜色，支持 CSS 变量或具体颜色值。

<solely-icon name="heart-fill" color="#ff4d4f"></solely-icon>
<solely-icon name="star-fill" color="#faad14"></solely-icon>
<solely-icon name="check-circle-fill" color="#52c41a"></solely-icon>
<solely-icon name="info-circle-fill" color="#1890ff"></solely-icon>
<solely-icon name="exclamation-triangle-fill" color="#fa8c16"></solely-icon>
<solely-icon name="x-circle-fill" color="#f5222d"></solely-icon>

图标旋转

使用 spin 属性开启旋转动画，支持顺时针和逆时针旋转。

<solely-icon name="arrow-clockwise" spin="clockwise" size="large"></solely-icon>
<solely-icon name="arrow-repeat" spin="counterclockwise" size="large"></solely-icon>
<solely-icon name="arrow-clockwise" spin="clockwise" size="large" spinDuration="2"></solely-icon>
<solely-icon name="gear-fill" spin="clockwise" size="large" color="#1890ff"></solely-icon>

CSS Part 自定义样式

通过 ::part() 伪元素可在 Shadow DOM 外部自定义图标内部样式，无需修改组件源码。

<!-- HTML -->
<solely-icon name="heart-fill" size="32"></solely-icon>

<!-- CSS：通过 ::part() 自定义样式 -->
<style>
  solely-icon::part(base) {
    filter: drop-shadow(0 2px 4px rgba(24, 144, 255, 0.4));
  }
  solely-icon[name="heart-fill"]::part(base) {
    color: #ff4d4f;
    transform: scale(1.2);
  }
</style>

API

属性

属性 说明 类型 默认值
name 图标名称 string -
size 图标尺寸，可选值：small、medium、large 或自定义数值 string | number medium
color 图标颜色，支持 CSS 变量或具体颜色值 string currentColor
spin 旋转方向，可选值：clockwise、counterclockwise string -
spinDuration 旋转一圈所需时间（秒） number 1
disabled 是否禁用 boolean false
sprite SVG Sprite 文件路径（优先级高于全局设置） string -

事件

事件名 说明 回调参数
click 点击图标时触发 (event: MouseEvent)

CSS Part

Part 名称 说明 对应元素
base 图标根元素（SVG） <svg>
placeholder 图标占位符（无图标时显示） <span>

静态方法

方法名 说明 参数
registerIcon 注册单个自定义图标 (name: string, svgContent: string)
registerIcons 批量注册自定义图标 (icons: Record<string, string>)
getIcon 获取已注册图标的 SVG 内容 (name: string) => string | undefined
hasIcon 检查图标是否已注册 (name: string) => boolean
getRegisteredIcons 获取所有已注册图标名称 () => string[]
setIconSpritePath 设置 SVG Sprite 文件路径（全局） (path: string) => void
loadIconSprite 手动加载 SVG Sprite 文件 (path: string) => Promise<void>

使用示例

// 导入图标组件和注册函数
import { SolelyIcon, registerIcon, setIconSpritePath, loadIconSprite } from 'solely';

// 方式一：全局设置 SVG Sprite（推荐）
setIconSpritePath('/icons/bootstrap-icons.svg');

// 方式二：手动注册自定义图标
registerIcon('my-icon', '<path d="M12 2L2 7l10 5 10-5-10-5z"/>');

// 使用图标
<solely-icon name="house-fill"></solely-icon>
