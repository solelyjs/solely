Timeline 时间轴

垂直展示的时间流信息，用于展示一系列信息按时间排列。

基础用法

最简单的用法，通过 items 属性传入时间轴数据。

<solely-timeline items='[{"title":"创建服务站点","time":"2024-01-01"},...]'></solely-timeline>

纵向模式

支持 left（左侧）、right（右侧）、alternate（交替）三种纵向布局。

左侧模式

右侧模式

<!-- 左侧模式 -->
<solely-timeline mode="left" items="..."></solely-timeline>

<!-- 右侧模式 -->
<solely-timeline mode="right" items="..."></solely-timeline>

交替模式

内容在时间轴两侧轮流出现。

<solely-timeline mode="alternate" items="[...]"></solely-timeline>

自定义颜色

通过 color 属性设置节点颜色，支持预设颜色或自定义颜色值。

// 预设颜色：blue、green、red、orange、gray
<solely-timeline items='[{"title":"蓝色","color":"blue"},...]'></solely-timeline>

Filled 变体

通过 variant="filled" 设置实心圆点样式。

<solely-timeline variant="filled" items="..."></solely-timeline>

自定义图标

通过 icon 属性设置自定义图标，支持 Emoji 或单字符图标。图标会以圆形背景包裹显示。

<solely-timeline items='[{"title":"项目启动","icon":"🚀","time":"2024-01-01"},...]'></solely-timeline>

待定状态 / Pending

设置 pending 属性显示最后一条虚线节点，带有流动动画效果。

<solely-timeline pending pending-content="持续记录中..." items="..."></solely-timeline>

Loading 状态

通过 item 的 loading 属性设置加载中的旋转动画。

<solely-timeline items='[{"title":"步骤一"}, {"title":"处理中...", "loading": true}]'></solely-timeline>

横向模式

支持 horizontal-top（顶部）、horizontal-bottom（底部）、horizontal-alternate（交替）三种横向布局。

顶部模式

底部模式

交替模式

<!-- 顶部模式 -->
<solely-timeline mode="horizontal-top" items="..."></solely-timeline>

<!-- 底部模式 -->
<solely-timeline mode="horizontal-bottom" items="..."></solely-timeline>

<!-- 交替模式 -->
<solely-timeline mode="horizontal-alternate" items="..."></solely-timeline>

倒序

设置 reverse 属性可使时间轴倒序显示。

<solely-timeline reverse items="..."></solely-timeline>

API

属性

属性名 说明 类型 默认值
mode 时间轴模式：left / right / alternate / horizontal-top / horizontal-bottom / horizontal-alternate string 'left'
items 时间轴数据（JSON字符串） string '[]'
variant 样式变体：outlined（空心圆点）/ filled（实心圆点） string 'outlined'
pending 是否显示待定节点（虚线圆点 + 流动连接线） boolean false
pendingContent 待定节点的内容文字 string '待定'
reverse 是否倒序排列节点 boolean false

Items 数据结构

字段 说明 类型
title 标题文字 string
content 详细内容 string
time 时间文字 string
color 颜色：blue / green / red / orange / gray 或自定义色值 string
icon 自定义图标（Emoji 或字符），以圆形背景包裹 string
loading 是否为 loading 状态（旋转动画） boolean
position 位置（仅 alternate 模式）：'left' | 'right' string
verticalPosition 垂直位置（仅 horizontal-alternate 模式）：'top' | 'bottom' string

方法

方法名 说明 参数 返回值
setItems(items) 动态设置时间轴项 items: TimelineItem[] void
