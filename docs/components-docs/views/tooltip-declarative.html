Tooltip 文字提示（声明式）

简单的文字提示气泡框，使用声明式方式编写，支持多种触发方式和位置。相比命令式组件，更符合组件化开发习惯。

基础用法

将需要添加提示的元素包裹在 <solely-tooltip> 组件中，通过 content 属性设置提示内容。

悬停我（上） 点击我（下） 
<!-- 悬停触发 -->
<solely-tooltip content="悬停显示提示" placement="top">
    <solely-button type="primary">悬停我</solely-button>
</solely-tooltip>

<!-- 点击触发 -->
<solely-tooltip content="点击显示提示" trigger="click" placement="bottom">
    <solely-button type="success">点击我</solely-button>
</solely-tooltip>

<!-- 聚焦触发 -->
<solely-tooltip content="聚焦显示提示" trigger="focus" placement="right">
    <input type="text" placeholder="聚焦我">
</solely-tooltip>

自定义颜色和宽度

通过 color 属性自定义背景色，通过 maxWidth 设置最大宽度。

红色提示 绿色提示 宽提示（300px） 
<!-- 自定义颜色 -->
<solely-tooltip content="红色警告" color="#ff4d4f">
    <solely-button type="error">红色提示</solely-button>
</solely-tooltip>

<!-- 自定义宽度 -->
<solely-tooltip 
    content="长文本..." 
    max-width="300">
    <solely-button>宽提示</solely-button>
</solely-tooltip>

包裹任意元素

Tooltip 可以包裹任意 HTML 元素，不仅限于按钮。

链接文字 ❓ Tag 标签 
<!-- 包裹链接 -->
<solely-tooltip content="链接提示">
    <a href="#">链接文字</a>
</solely-tooltip>

<!-- 包裹图标 -->
<solely-tooltip content="图标提示">
    <span>❓</span>
</solely-tooltip>

<!-- 包裹图片 -->
<solely-tooltip content="图片说明">
    <img src="..." alt="图片">
</solely-tooltip>

手动控制（方法调用）

可以通过 JavaScript 调用组件的 show() 和 hide() 方法来手动控制提示的显示与隐藏。

目标按钮
调用 show() 调用 hide()

<!-- HTML 模板 -->
<solely-tooltip 
    id="my-tooltip"
    content="手动控制的提示"
    trigger="manual">
    <solely-button>目标按钮</solely-button>
</solely-tooltip>

<button @click="showTooltip()">显示</button>
<button @click="hideTooltip()">隐藏</button>

// JavaScript
const tooltip = document.querySelector('#my-tooltip');
tooltip.show();   // 显示提示
tooltip.hide();   // 隐藏提示

CSS Part 自定义样式

通过 ::part() 伪元素可在 Shadow DOM 外部自定义 Tooltip 内部样式，无需修改组件源码。

悬停查看自定义样式 
<!-- HTML -->
<solely-tooltip content="自定义渐变样式" placement="top">
  <solely-button type="primary">悬停查看自定义样式</solely-button>
</solely-tooltip>

<!-- CSS：通过 ::part() 自定义样式 -->
<style>
  solely-tooltip::part(base) {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    border-radius: 8px;
  }
  solely-tooltip::part(arrow) {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  }
  solely-tooltip::part(content) {
    color: #fff;
    font-weight: 600;
  }
</style>

API

属性

属性 说明 类型 默认值
content 提示内容 string ''
placement 气泡框位置 'top' | 'bottom' | 'left' | 'right' | 'topLeft' | 'topRight' | 'bottomLeft' | 'bottomRight' | 'leftTop' | 'leftBottom' | 'rightTop' | 'rightBottom' 'top'
trigger 触发方式 'hover' | 'click' | 'focus' | 'manual' 'hover'
visible 是否显示（受控模式） boolean false
color 背景颜色 string -
maxWidth 最大宽度（像素） number 250

方法

方法名 说明 参数 返回值
show() 显示提示 - void
hide() 隐藏提示 - void

CSS Part

Part 名称 说明 对应元素
wrapper Tooltip 包装器根元素 <div>
base Tooltip 气泡框主体 <div>
content 提示文本内容区 <div>
arrow 气泡框箭头 <div>

类型定义

// TooltipPlacement 类型
type TooltipPlacement =
    | 'top'
    | 'bottom'
    | 'left'
    | 'right'
    | 'topLeft'
    | 'topRight'
    | 'bottomLeft'
    | 'bottomRight'
    | 'leftTop'
    | 'leftBottom'
    | 'rightTop'
    | 'rightBottom';

// TooltipTrigger 类型
type TooltipTrigger = 'hover' | 'click' | 'focus' | 'manual';

// TooltipProps 接口
interface TooltipProps {
    content?: string;           // 提示内容
    placement?: TooltipPlacement;   // 气泡框位置
    trigger?: TooltipTrigger;       // 触发方式
    visible?: boolean;            // 是否显示（受控模式）
    color?: string;             // 背景颜色
    maxWidth?: number;          // 最大宽度（像素）
}
