Breadcrumb 面包屑

显示当前页面在系统层级结构中的位置，并能向上返回。

基础用法

最简单的面包屑，使用 items 属性设置路由栈。

// HTML
<solely-breadcrumb id="breadcrumb"></solely-breadcrumb>

// JavaScript
const breadcrumb = document.getElementById('breadcrumb');
const items = [
    { title: '首页', href: '/' },
    { title: '组件', href: '/components' },
    { title: '面包屑' }
];
breadcrumb.setAttribute('items', JSON.stringify(items));

自定义分隔符

使用 separator 属性自定义分隔符。

// 使用箭头分隔符
breadcrumb.setAttribute('separator', '>');

// 使用竖线分隔符
breadcrumb.setAttribute('separator', '|');

// 使用箭头符号
breadcrumb.setAttribute('separator', '→');

点击事件

为面包屑项添加点击事件处理。

点击面包屑项查看事件
{{$data.title}} 
// HTML
<solely-breadcrumb id="breadcrumb"></solely-breadcrumb>

// JavaScript
const breadcrumb = document.getElementById('breadcrumb');
const items = [
    { 
        title: '首页', 
        onClick: () => console.log('点击了首页')
    },
    { 
        title: '组件', 
        onClick: () => console.log('点击了组件')
    },
    { title: '面包屑' }
];
breadcrumb.setAttribute('items', JSON.stringify(items));

// 监听点击事件
breadcrumb.addEventListener('click', () => {
    const item = breadcrumb.clickedItem;
    console.log('点击了:', item?.title);
});

使用链接

为面包屑项添加 href 属性，使其成为可点击的链接。

const items = [
    { title: '首页', href: '/' },
    { title: '产品中心', href: '/products' },
    { title: '详情' }
];
breadcrumb.setAttribute('items', JSON.stringify(items));

API

属性

属性 说明 类型 默认值
items 面包屑项列表（JSON 字符串） string '[]'
separator 分隔符 string '/'

事件

事件名 说明 回调参数
click 点击面包屑项时触发（无 href 时） 通过 breadcrumb.clickedItem 获取

属性

属性名 说明 类型
items 获取/设置面包屑项 BreadcrumbItem[]
clickedItem 获取最后点击的面包屑项 BreadcrumbItem | null

类型定义

// 面包屑项
interface BreadcrumbItem {
  title: string;           // 显示标题（必填）
  href?: string;          // 跳转链接（可选）
  onClick?: () => void;  // 点击回调（可选）
}

// 面包屑属性
interface BreadcrumbProps {
  items: string;         // JSON 字符串格式的 BreadcrumbItem[]
  separator: string;     // 分隔符，默认 '/'
}
