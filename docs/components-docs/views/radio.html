Radio 单选框

单选框组件，用于在多个选项中进行单选。

基础用法

最简单的用法，通过 checked 属性控制选中状态，使用相同的 name 属性来分组。

Apple Banana Orange 
<solely-radio name="fruit" value="apple" checked>Apple</solely-radio>
<solely-radio name="fruit" value="banana">Banana</solely-radio>
<solely-radio name="fruit" value="orange">Orange</solely-radio>

禁用状态

通过 disabled 属性禁用单选框，禁用后无法交互。

Disabled Disabled Checked 
<solely-radio disabled>Disabled</solely-radio>
<solely-radio disabled checked>Disabled Checked</solely-radio>

颜色类型

通过 type 属性设置不同的颜色主题，可选 primary、success、warning、error。

Primary Success Warning Error 
<solely-radio type="primary" checked>Primary</solely-radio>
<solely-radio type="success" checked>Success</solely-radio>
<solely-radio type="warning" checked>Warning</solely-radio>
<solely-radio type="error" checked>Error</solely-radio>

块级模式

通过 block 属性使单选框占满父容器宽度，适用于表单布局场景。

选项一（block 模式） 选项二（block 模式） 选项三（block 模式） 
<solely-radio name="block-demo" value="option1" block>选项一（block 模式）</solely-radio>
<solely-radio name="block-demo" value="option2" block>选项二（block 模式）</solely-radio>
<solely-radio name="block-demo" value="option3" block>选项三（block 模式）</solely-radio>

事件处理

监听 change 事件获取选中的值。点击单选框查看效果。

Pizza Burger Sushi 选中: {{$data.selectedValue}} 
<!-- HTML -->
<solely-radio name="food" value="pizza" @change="handleChange">Pizza</solely-radio>
<solely-radio name="food" value="burger" @change="handleChange">Burger</solely-radio>
<span>选中: {{$data.selectedValue}}</span>

<!-- JavaScript -->
const radioEls = document.querySelectorAll('solely-radio[name="food"]');

radioEls.forEach(radio => {
  radio.addEventListener('change', (e) => {
    const target = e.target;
    const value = target.value;
    const checked = target.checked;
    if (checked) {
      console.log(`选中: ${value}`);
    }
  });
});

双向绑定

使用 s-model 指令实现双向数据绑定，绑定的是 checked 属性。适用于需要程序化控制选中状态的场景。

选项 A 当前状态: {{$data.modelRadio1 ? '已选中' : '未选中'}}
选中 取消 切换 
<!-- 使用 s-model 实现双向绑定（绑定 checked 属性） -->
<solely-radio s-model="isSelected">
  选项 A
</solely-radio>

<!-- JavaScript -->
// s-model 会自动绑定 checked 属性和监听 change 事件
// solely-radio 的 model 配置: { prop: 'checked', event: 'change' }

// 可以通过代码控制选中状态
this.isSelected = true;   // 选中
this.isSelected = false;  // 取消选中

API

属性

属性 说明 类型 默认值
checked 指定当前是否选中（支持属性和方法操作） boolean false
disabled 是否禁用 boolean false
content 单选框文本内容 string ''
value 单选框的值，用于标识选项（支持属性和方法操作） string ''
name 单选框名称，用于分组，同组的单选框互斥 string ''
type 颜色类型，可选 primary、success、warning、error string 'primary'
block 是否块级元素，占满父容器宽度 boolean false

事件

事件名 说明 回调参数
change 选中状态变化时触发 Event

方法

方法名 说明 参数 返回值
setChecked(checked) 设置选中状态 checked: boolean void
getChecked() 获取当前选中状态 - boolean

类型定义

// 单选框颜色类型
type RadioType = 'primary' | 'success' | 'warning' | 'error';

// 单选框属性接口
interface RadioProps {
  checked?: boolean;   // 是否选中
  disabled?: boolean;  // 是否禁用
  content?: string;   // 文本内容
  value?: string;     // 单选框值
  name?: string;      // 分组名称
  type?: RadioType;   // 颜色类型
  block?: boolean;   // 是否块级元素
}
