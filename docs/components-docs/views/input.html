Input 输入框

通过鼠标或键盘输入内容，是最基础的表单域的包装。

基础用法

基本的输入框，支持 placeholder 提示。

<solely-input placeholder="请输入内容"></solely-input>

输入框尺寸

支持 small、medium（默认）、large 三种尺寸。

<solely-input size="small" placeholder="Small"></solely-input>
<solely-input size="medium" placeholder="Medium"></solely-input>
<solely-input size="large" placeholder="Large"></solely-input>

可清空

设置 clearable 属性，输入内容后会显示清空按钮。

<solely-input clearable placeholder="输入后可清空"></solely-input>

密码输入框

设置 type="password" 为密码输入框，配合 showpassword 可切换显示隐藏。

<solely-input type="password" placeholder="请输入密码"></solely-input>
<solely-input type="password" showpassword placeholder="可切换显示"></solely-input>

禁用状态

设置 disabled 属性禁用输入框。

<solely-input disabled placeholder="禁用状态"></solely-input>
<solely-input disabled value="已禁用的值"></solely-input>

只读状态

设置 readonly 属性使输入框只读。

<solely-input readonly value="只读内容"></solely-input>

前缀和后缀

通过 prefix 和 suffix 属性设置前缀和后缀图标或文字。

<solely-input prefix="￥" placeholder="金额"></solely-input>
<solely-input suffix="@qq.com" placeholder="邮箱"></solely-input>
<solely-input prefix="🔍" placeholder="搜索"></solely-input>

Prefix/Suffix 插槽

使用 prefix 和 suffix 插槽可以自定义更复杂的前缀和后缀内容。

$ ✓ 📧 .com 
<solely-input placeholder="使用插槽前缀">
  <span slot="prefix" style="color: var(--solely-primary); font-weight: bold;">$</span>
</solely-input>

<solely-input placeholder="使用插槽后缀">
  <span slot="suffix" style="color: var(--solely-success);">✓</span>
</solely-input>

<solely-input placeholder="同时使用前后缀插槽">
  <span slot="prefix">📧</span>
  <span slot="suffix" style="color: var(--solely-text-tertiary);">.com</span>
</solely-input>

输入框类型

支持多种原生输入类型。

<solely-input type="text" placeholder="文本"></solely-input>
<solely-input type="number" placeholder="数字"></solely-input>
<solely-input type="email" placeholder="邮箱"></solely-input>
<solely-input type="tel" placeholder="电话"></solely-input>
<solely-input type="url" placeholder="网址"></solely-input>
<solely-input type="search" placeholder="搜索"></solely-input>

文本域

设置 type="textarea" 使用多行文本输入。

<solely-input type="textarea" placeholder="请输入多行文本"></solely-input>
<solely-input type="textarea" rows="5" placeholder="5行高度"></solely-input>

字数统计

设置 showcount 和 maxlength 显示字数统计。

<solely-input showcount maxlength="100" placeholder="最多输入100个字符"></solely-input>

状态样式

设置 status 显示不同的状态样式。

<solely-input status="error" message="请输入正确的邮箱格式"></solely-input>
<solely-input status="warning" message="密码强度较弱"></solely-input>
<solely-input status="success" message="验证通过"></solely-input>

输入格式化

设置 format 自动格式化输入内容。

<solely-input format="phone" placeholder="手机号自动分段"></solely-input>
<solely-input format="bankcard" placeholder="银行卡号自动分段"></solely-input>

复制功能

设置 copyable 显示复制按钮。

<solely-input copyable value="点击右侧按钮复制我"></solely-input>

自动聚焦

设置 autofocus 属性，组件挂载后自动获得焦点。

<solely-input autofocus placeholder="页面加载后自动聚焦"></solely-input>

聚焦全选

设置 selectall 属性，聚焦时自动全选内容。

<solely-input selectall value="点击输入框自动全选这段文字"></solely-input>

搜索建议

设置 suggestions 属性启用搜索建议功能，输入时会自动过滤匹配的建议。

<!-- 直接在 HTML 中设置建议数据 -->
<solely-input suggestions="['Apple', 'Banana', 'Cherry']" placeholder="输入关键词搜索..."></solely-input>

<!-- 或通过 JavaScript 动态设置 -->
const input = document.querySelector('solely-input');
input.suggestions = ['Apple', 'Banana', 'Cherry', 'Date'];

块级输入框

添加 block 属性可以让输入框宽度占满父容器。

<solely-input block placeholder="块级输入框，宽度占满父容器"></solely-input>
<solely-input block size="large" placeholder="大尺寸块级输入框"></solely-input>

事件监听

监听 input、change、focus、blur 等事件来响应输入框的交互。

事件日志（最近5条）：
等待输入...
<!-- 监听所有事件 -->
<solely-input
  @input="handleInput(event)"
  @change="handleChange(event)"
  @focus="handleFocus(event)"
  @blur="handleBlur(event)"
  @keydown="handleKeydown(event)"
  @clear="handleClear(event)"
  @search="handleSearch(event)"
  @copy="handleCopy(event)"
  @copy-error="handleCopyError(event)"
></solely-input>

// 原生事件：input、change、focus、blur、keydown
// 通过 event.target.value 获取值
function handleInput(event: InputEvent) {
  const target = event.target as HTMLInputElement;
  console.log('input:', target.value);
}

// change 事件通过 Event 派发，通过 event.target.value 获取值
function handleChange(event: Event) {
  const target = event.target as HTMLInputElement;
  console.log('change:', target.value);
}

function handleFocus(event: FocusEvent) {
  console.log('focus');
}

function handleBlur(event: FocusEvent) {
  console.log('blur');
}

function handleKeydown(event: KeyboardEvent) {
  console.log('keydown:', event.key);
}

// 自定义事件：clear、search、copy、copy-error
function handleClear(event: Event) {
  console.log('clear');
}

function handleSearch(event: CustomEvent<{ value: string }>) {
  console.log('search:', event.detail.value);
}

function handleCopy(event: CustomEvent<{ value: string }>) {
  console.log('copy:', event.detail.value);
}

function handleCopyError(event: CustomEvent<{ error: Error }>) {
  console.log('copy-error:', event.detail.error);
}

// 通过属性获取/设置值
const input = document.querySelector('solely-input');
console.log(input.value); // 获取值
input.value = '新值'; // 设置值

双向绑定

使用 s-model 指令实现双向数据绑定，绑定的是 value 属性。

绑定值：
{{$data.modelInput1 || '（空）'}}
密码值：
{{$data.modelInput2 || '（空）'}}
文本域值：
{{$data.modelInput3 || '（空）'}}
清空所有 
<!-- 使用 s-model 实现双向绑定（绑定 value 属性） -->
<solely-input s-model="textValue" placeholder="输入内容"></solely-input>

<!-- 密码输入框 -->
<solely-input type="password" s-model="passwordValue"></solely-input>

<!-- 文本域 -->
<solely-input type="textarea" s-model="textareaValue"></solely-input>

<!-- JavaScript -->
// s-model 会自动绑定 value 属性和监听 change 事件
// 未配置 model 的组件默认使用: { prop: 'value', event: 'change' }

API

属性

属性 说明 类型 默认值
value 输入框值（支持属性和方法操作） string ''
placeholder 占位提示文字 string -
type 输入框类型 'text' | 'password' | 'number' | 'email' | 'tel' | 'url' | 'search' | 'textarea' 'text'
size 输入框尺寸 'small' | 'medium' | 'large' 'medium'
disabled 是否禁用 boolean false
readonly 是否只读 boolean false
required 是否必填 boolean false
maxlength 最大长度 number -
minlength 最小长度 number -
clearable 是否可清空 boolean false
showpassword 是否显示密码切换按钮 boolean false
prefix 前缀图标或文字 string -
suffix 后缀图标或文字 string -
status 状态 'error' | 'warning' | 'success' | '' ''
message 状态提示信息 string -
showcount 是否显示字数统计 boolean false
rows 文本域行数 number 3
autofocus 是否自动聚焦 boolean false
selectall 聚焦时是否全选 boolean false
copyable 是否显示复制按钮 boolean false
format 输入格式化类型 'none' | 'phone' | 'bankcard' | 'idcard' 'none'
suggestions 搜索建议数据源（设置后自动启用搜索建议） string[] []
block 是否块级显示，设置为 true 时宽度占满父容器 boolean false

事件

事件名 说明 回调参数
input 输入时触发（原生事件） InputEvent
change 值改变时触发（失焦或回车） Event
focus 获得焦点时触发（原生事件） FocusEvent
blur 失去焦点时触发（原生事件） FocusEvent
keydown 键盘按下时触发（原生事件） KeyboardEvent
clear 点击清空按钮时触发 Event
search 搜索框回车时触发（自定义事件） CustomEvent<{ value: string }>
copy 复制成功时触发（自定义事件） CustomEvent<{ value: string }>
copy-error 复制失败时触发（自定义事件） CustomEvent<{ error: Error }>

插槽

插槽名 说明
prefix 输入框前缀内容（优先级高于 prefix 属性）
suffix 输入框后缀内容（优先级高于 suffix 属性，但低于 clearable、showpassword、copyable）

方法

方法名 说明
focus() 使输入框获得焦点
blur() 使输入框失去焦点
select() 选中输入框中的文字
setValue(value) 设置输入框的值
getValue() 获取输入框的值（去除格式化空格）

类型定义

// 输入框类型
type InputType =
  | 'text'
  | 'password'
  | 'number'
  | 'email'
  | 'tel'
  | 'url'
  | 'search'
  | 'textarea';

// 输入框尺寸
type InputSize = 'small' | 'medium' | 'large';

// 输入框属性接口
interface InputProps {
  type?: InputType;           // 输入框类型
  size?: InputSize;           // 输入框尺寸
  value?: string;            // 输入框值
  placeholder?: string;      // 占位符
  disabled?: boolean;        // 是否禁用
  readonly?: boolean;        // 是否只读
  clearable?: boolean;       // 是否可清空
  showcount?: boolean;       // 是否显示字数统计
  maxlength?: number;        // 最大长度
  minlength?: number;        // 最小长度
  prefix?: string;           // 前缀
  suffix?: string;           // 后缀
  rows?: number;             // 文本域行数
  cols?: number;             // 文本域列数
  allowcopy?: boolean;       // 是否允许复制
  copytips?: string;         // 复制提示文本
  block?: boolean;          // 是否块级显示
}
