快速上手

了解如何在项目中使用 Solely UI。

安装

UI 组件库随着 Solely 框架一起分发，只需安装 Solely 即可获得全套组件。

# npm
npm install solely

# yarn
yarn add solely

# pnpm
pnpm add solely

引入组件

你可以按需引入单个组件，也可以引入所有组件。

// 按需引入单个组件（推荐）
import 'solely/components/button';
import 'solely/components/input';
import 'solely/components/card';

// 或者引入所有组件
import 'solely/components';

引入样式

组件库需要单独引入 CSS 主题文件，支持亮色/暗色模式。

// 引入主题 CSS 变量（必需）
import 'solely/components/theme/variables.css';

// 按需引入组件样式
import 'solely/components/elements/button/style.css';
import 'solely/components/elements/input/style.css';

使用组件

在 HTML 中直接使用组件标签。

<!-- 基础按钮 -->
<solely-button type="primary">Primary Button</solely-button>

<!-- 输入框 -->
<solely-input placeholder="请输入内容"></solely-input>

<!-- 带图标的按钮 -->
<solely-button type="primary" icon="search">
    搜索
</solely-button>

命令式组件

Solely UI 提供了一系列命令式组件，可以通过 JavaScript 直接调用。

import { Message, Modal, Drawer } from 'solely/components/commands';

// Message 全局提示
Message.success('操作成功！');
Message.error('操作失败！');
Message.warning('警告信息');
Message.info('提示信息');

// Modal 对话框
Modal.confirm({
    title: '确认删除',
    content: '确定要删除此项吗？',
    onOk: () => {
        console.log('用户确认');
    }
});

// Drawer 抽屉
const drawer = Drawer.open({
    title: '抽屉标题',
    content: '抽屉内容',
    placement: 'right'
});

// 关闭抽屉
drawer.close();

TypeScript 支持

Solely UI 提供完整的 TypeScript 类型定义。

import { Message, Modal } from 'solely/components/commands';
import type { MessageOptions, ModalOptions } from 'solely/components/commands';

// 使用类型定义
const messageOptions: MessageOptions = {
    content: '提示信息',
    duration: 3000
};

const modalOptions: ModalOptions = {
    title: '标题',
    content: '内容',
    onOk: () => {
        console.log('确定');
    }
};

Message.open(messageOptions);
Modal.open(modalOptions);

主题定制

通过 CSS 变量自定义主题。

/* 自定义主题变量 */
:root {
  --solely-primary: #1890ff;
  --solely-success: #52c41a;
  --solely-warning: #faad14;
  --solely-error: #f5222d;
  --solely-font-size-md: 14px;
  --solely-border-radius-md: 6px;
}

/* 暗黑模式 */
[data-theme="dark"] {
  --solely-bg-color: #141414;
  --solely-component-bg: #1f1f1f;
}

主题样式

引入全部组件时，主题 CSS 变量会自动加载；按需引入时需要单独导入主题样式。

// 引入全部组件（主题 CSS 已包含）
import 'solely/components';

// 按需引入时需要先导入主题 CSS
import 'solely/components/theme/variables.css';
import 'solely/components/button';

// 使用主题切换功能
import { setTheme, toggleTheme } from 'solely/components/theme';
toggleTheme(); // 切换明暗主题
