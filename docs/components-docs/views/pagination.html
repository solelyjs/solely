Pagination 分页

用于数据分页展示，支持页码切换、快速跳转、每页条数设置等功能。

基础用法

最简单的用法，通过 current 设置当前页，total 设置数据总数。

<solely-pagination
  current="1"
  total="50">
</solely-pagination>

显示总数

设置 show-total 属性显示数据总数。

<solely-pagination
  current="1"
  total="100"
  show-total>
</solely-pagination>

快速跳转

设置 show-quick-jumper 属性显示快速跳转输入框。

<solely-pagination
  current="1"
  total="100"
  show-quick-jumper>
</solely-pagination>

每页条数选择

设置 show-size-changer 属性显示每页条数选择器。

<solely-pagination
  current="1"
  total="100"
  show-size-changer>
</solely-pagination>

小尺寸

设置 size="small" 使用小尺寸分页。

<solely-pagination
  current="1"
  total="50"
  size="small">
</solely-pagination>

禁用状态

设置 disabled 属性禁用分页组件。

<solely-pagination
  current="1"
  total="50"
  disabled>
</solely-pagination>

大量数据

当数据量较大时，会自动折叠页码显示。

<solely-pagination
  current="5"
  total="1000"
  show-total
  show-quick-jumper>
</solely-pagination>

完整功能

展示分页组件的完整功能组合。

<solely-pagination
  current="1"
  total="100"
  show-total
  show-size-changer
  show-quick-jumper>
</solely-pagination>

CSS Part 自定义样式

通过 ::part() 伪元素可在 Shadow DOM 外部分页组件内部样式，无需修改组件源码。

<!-- HTML -->
<solely-pagination current="3" total="100" show-total></solely-pagination>

<!-- CSS：通过 ::part() 自定义样式 -->
<style>
  solely-pagination::part(item) {
    border-radius: 8px;
    border: 1px solid #d9d9d9;
    transition: all 0.2s;
  }
  solely-pagination::part(item):hover {
    border-color: #1890ff;
    color: #1890ff;
  }
  solely-pagination::part(item-active) {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: #fff;
    box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);
  }
  solely-pagination::part(total) {
    color: #666;
    font-weight: 500;
  }
</style>

事件演示

切换页码或每页条数查看事件触发情况。

事件日志（最近5条）：
等待交互...
<!-- HTML -->
<solely-pagination
  current="1"
  total="100"
  @change="handleChange(event)"
  @sizeChange="handleSizeChange(event)">
</solely-pagination>

<!-- JavaScript -->
const pagination = document.querySelector('solely-pagination');

// 页码改变事件
pagination.addEventListener('change', (e) => {
  console.log('当前页:', e.detail.current);
  console.log('每页条数:', e.detail.pageSize);
});

// 每页条数改变事件（兼容旧名称 showSizeChange）
pagination.addEventListener('sizeChange', (e) => {
  console.log('当前页:', e.detail.current);
  console.log('每页条数:', e.detail.pageSize);
});

API

属性

属性名 说明 类型 默认值
current 当前页码 number 1
total 数据总数 number 0
pageSize 每页条数 number 10
size 组件尺寸，可选值为 default、small string default
showTotal 是否显示总数据数 boolean false
showQuickJumper 是否显示快速跳转 boolean false
showSizeChanger 是否显示每页条数选择器 boolean false
pageSizeOptions 每页条数选项（JSON字符串） string [10, 20, 50, 100]
disabled 是否禁用 boolean false

事件

事件名 说明 回调参数
change 页码改变时触发 CustomEvent<{ current: number; pageSize: number }>
sizeChange 每页条数改变时触发（兼容旧名称 showSizeChange） CustomEvent<{ current: number; pageSize: number }>

CSS Part

Part 名称 说明 对应元素
base 分页组件根容器 <div>
total 总数文本 <span>
prev 上一页按钮 <span>
item 页码按钮 <span>
ellipsis 省略号 <span>
next 下一页按钮 <span>
size-changer 每页条数选择器 <span>
jumper 快速跳转区域 <span>
jumper-input-wrapper 跳转输入框包装器 <span>
jumper-input 跳转输入框 <input>
jumper-controls 跳转控制按钮容器 <span>
jumper-up 跳转增加按钮 <span>
jumper-down 跳转减少按钮 <span>

类型定义

// 分页尺寸
type PaginationSize = 'default' | 'small';

// 分页属性接口
interface PaginationProps {
  current?: number;           // 当前页码
  total?: number;             // 数据总数
  pageSize?: number;          // 每页条数
  size?: PaginationSize;    // 组件尺寸
  showTotal?: boolean;       // 是否显示总数
  showQuickJumper?: boolean; // 是否显示快速跳转
  showSizeChanger?: boolean; // 是否显示条数选择
  disabled?: boolean;        // 是否禁用
}
