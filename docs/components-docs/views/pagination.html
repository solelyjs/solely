Pagination 分页

用于数据分页展示，支持页码切换、快速跳转、每页条数设置等功能。

基础用法

最简单的用法，通过 current 设置当前页，total 设置数据总数。

<solely-pagination
  current="1"
  total="50">
</solely-pagination>

显示总数

设置 show-total 属性显示数据总数。

<solely-pagination
  current="1"
  total="100"
  show-total>
</solely-pagination>

快速跳转

设置 show-quick-jumper 属性显示快速跳转输入框。

<solely-pagination
  current="1"
  total="100"
  show-quick-jumper>
</solely-pagination>

每页条数选择

设置 show-size-changer 属性显示每页条数选择器。

<solely-pagination
  current="1"
  total="100"
  show-size-changer>
</solely-pagination>

小尺寸

设置 size="small" 使用小尺寸分页。

<solely-pagination
  current="1"
  total="50"
  size="small">
</solely-pagination>

禁用状态

设置 disabled 属性禁用分页组件。

<solely-pagination
  current="1"
  total="50"
  disabled>
</solely-pagination>

大量数据

当数据量较大时，会自动折叠页码显示。

<solely-pagination
  current="5"
  total="1000"
  show-total
  show-quick-jumper>
</solely-pagination>

完整功能

展示分页组件的完整功能组合。

<solely-pagination
  current="1"
  total="100"
  show-total
  show-size-changer
  show-quick-jumper>
</solely-pagination>

事件演示

切换页码或每页条数查看事件触发情况。

<!-- HTML -->
<solely-pagination
  current="1"
  total="100"
  @change="handleChange(event)"
  @showSizeChange="handleSizeChange(event)">
</solely-pagination>

<!-- JavaScript -->
const pagination = document.querySelector('solely-pagination');

// 页码改变事件
pagination.addEventListener('change', (e) => {
  console.log('当前页:', e.detail.current);
  console.log('每页条数:', e.detail.pageSize);
});

// 每页条数改变事件
pagination.addEventListener('showSizeChange', (e) => {
  console.log('当前页:', e.detail.current);
  console.log('每页条数:', e.detail.pageSize);
});

API

属性

属性名 说明 类型 默认值
current 当前页码 number 1
total 数据总数 number 0
pageSize 每页条数 number 10
size 组件尺寸，可选值为 default、small string default
showTotal 是否显示总数据数 boolean false
showQuickJumper 是否显示快速跳转 boolean false
showSizeChanger 是否显示每页条数选择器 boolean false
pageSizeOptions 每页条数选项（JSON字符串） string [10, 20, 50, 100]
disabled 是否禁用 boolean false

事件

事件名 说明 回调参数
change 页码改变时触发 CustomEvent<{ current: number; pageSize: number }>
showSizeChange 每页条数改变时触发 CustomEvent<{ current: number; pageSize: number }>

方法

方法名 说明 参数
setCurrent(current) 设置当前页 current: number
getCurrent() 获取当前页 -

类型定义

// 分页尺寸
type PaginationSize = 'default' | 'small';

// 分页属性接口
interface PaginationProps {
  current?: number;           // 当前页码
  total?: number;             // 数据总数
  pageSize?: number;          // 每页条数
  size?: PaginationSize;    // 组件尺寸
  showTotal?: boolean;       // 是否显示总数
  showQuickJumper?: boolean; // 是否显示快速跳转
  showSizeChanger?: boolean; // 是否显示条数选择
  disabled?: boolean;        // 是否禁用
}
