Rate 评分

评分组件，用于展示或设置评分。

基础用法

最简单的用法。

<solely-rate></solely-rate>

设置初始值

通过 value 属性设置初始评分值。

<solely-rate value="3"></solely-rate>
<solely-rate value="4.5"></solely-rate>

星星数量

通过 count 属性设置星星总数，默认为 5。

<solely-rate count="3"></solely-rate>
<solely-rate count="5"></solely-rate>
<solely-rate count="10"></solely-rate>

允许半选

设置 allow-half 属性允许选择半星。

<solely-rate allow-half value="2.5"></solely-rate>
<solely-rate allow-half value="3.5"></solely-rate>

自定义字符

通过 character 属性自定义评分图标。

<solely-rate character="❤"></solely-rate>
<solely-rate character="✦"></solely-rate>
<solely-rate character="★"></solely-rate>
<solely-rate character="A"></solely-rate>

允许清除

设置 allow-clear 属性，再次点击相同评分可清除。

<solely-rate allow-clear value="3"></solely-rate>

禁用状态

通过 disabled 属性禁用评分。

<solely-rate disabled value="3"></solely-rate>
<solely-rate disabled value="4.5" allow-half></solely-rate>

只读状态

通过 readonly 属性设置只读状态，用于展示评分。

4.5 分 
<solely-rate readonly value="4.5" allow-half></solely-rate>

API

属性

属性名 说明 类型 默认值
value 当前评分值 number 0
count 星星总数 number 5
allowHalf 是否允许半选 boolean false
disabled 是否禁用 boolean false
allowClear 是否允许清除 boolean false
character 自定义字符 string ★
readonly 是否只读 boolean false

事件

事件名 说明 回调参数
change 评分改变时触发 Event

方法

方法名 说明 参数 返回值
setValue(value) 设置评分值 value: number void
getValue() 获取当前评分值 - number

类型定义

// 组件属性接口
interface RateProps {
  value?: number;       // 当前评分值
  count?: number;       // 星星总数
  allowHalf?: boolean;  // 是否允许半选
  disabled?: boolean;   // 是否禁用
  allowClear?: boolean; // 是否允许清除
  character?: string;  // 自定义字符
  readonly?: boolean;  // 是否只读
}
