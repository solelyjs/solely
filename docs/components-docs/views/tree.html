Tree 树形控件

树形控件，用于展示具有层级关系的数据。

基础用法

最简单的用法，通过 tree-data 属性传入树形数据。

<solely-tree 
  tree-data='[{"title":"父节点 1","key":"0-0","children":[...]},...]'
></solely-tree>

连接线

设置 show-line 属性显示节点之间的连接线。

<solely-tree show-line tree-data="..."></solely-tree>

默认展开全部

设置 default-expand-all 属性默认展开所有节点。

<solely-tree default-expand-all tree-data="..."></solely-tree>

多选

设置 multiple 属性支持多选节点。

<solely-tree multiple tree-data="..."></solely-tree>

可勾选

设置 checkable 属性显示勾选框。

<solely-tree checkable tree-data="..."></solely-tree>

禁用状态

设置 disabled 属性禁用整个树形控件，或在节点数据中设置 disabled 禁用特定节点。

<!-- 禁用整个树 -->
<solely-tree disabled tree-data="..."></solely-tree>

<!-- 禁用特定节点 -->
<solely-tree tree-data='[{"title":"父节点","key":"0-0","children":[{"title":"禁用节点","key":"0-0-1","disabled":true}]}]'></solely-tree>

Block 属性

设置 block 属性可使树形控件宽度撑满父容器。

<solely-tree block tree-data="..."></solely-tree>

事件处理

监听 select、expand、check 事件处理树形控件的交互。

<!-- HTML -->
<solely-tree 
  multiple
  checkable
  @select="handleSelect(event)"
  @expand="handleExpand(event)"
  @check="handleCheck(event)"
  tree-data="..."
></solely-tree>

<!-- JavaScript -->
const treeEl = document.querySelector('solely-tree');

// select 事件 - 选中节点时触发
treeEl.addEventListener('select', (e) => {
  const { node, selectedKeys } = e.detail;
  console.log('选中节点:', node, '选中 keys:', selectedKeys);
});

// expand 事件 - 展开/收起节点时触发
treeEl.addEventListener('expand', (e) => {
  const { node, expandedKeys } = e.detail;
  console.log('展开节点:', node, '展开 keys:', expandedKeys);
});

// check 事件 - 勾选节点时触发
treeEl.addEventListener('check', (e) => {
  const { node, checkedKeys } = e.detail;
  console.log('勾选节点:', node, '勾选 keys:', checkedKeys);
});

API

属性

属性名 说明 类型 默认值
treeData 树形数据（JSON字符串） string []
showLine 是否显示连接线 boolean false
multiple 是否支持多选 boolean false
checkable 是否显示勾选框 boolean false
defaultExpandAll 是否默认展开所有节点 boolean false
disabled 是否禁用 boolean false
searchable 是否可搜索 boolean false
searchValue 搜索值 string -
block 是否撑满父容器宽度 boolean false

事件

事件名 说明 回调参数
select 选中节点时触发 CustomEvent<{ node: TreeNode; selectedKeys: string[] }>
expand 展开/收起节点时触发 CustomEvent<{ node: TreeNode; expandedKeys: string[] }>
check 勾选节点时触发 CustomEvent<{ node: TreeNode; checkedKeys: string[] }>

方法

方法名 说明 参数 返回值
setTreeData(data) 设置树形数据 data: TreeNode[] void
getSelectedKeys() 获取选中的 keys - string[]
getCheckedKeys() 获取勾选的 keys - string[]
getExpandedKeys() 获取展开的 keys - string[]
expandAll() 展开所有节点 - void
collapseAll() 收起所有节点 - void

类型定义

// 树形节点
interface TreeNode {
  title: string;        // 节点标题
  key: string;          // 节点唯一标识
  children?: TreeNode[]; // 子节点
  disabled?: boolean;    // 是否禁用
  selectable?: boolean; // 是否可选中
  checkable?: boolean;  // 是否可勾选
  isLeaf?: boolean;     // 是否为叶子节点
  icon?: string;       // 自定义图标
}

// 组件属性接口
interface TreeProps {
  treeData?: string;     // 树形数据（JSON字符串）
  showLine?: boolean;   // 是否显示连接线
  multiple?: boolean;   // 是否支持多选
  checkable?: boolean;  // 是否显示勾选框
  defaultExpandAll?: boolean; // 是否默认展开所有节点
  disabled?: boolean;   // 是否禁用
  searchable?: boolean; // 是否可搜索
  searchValue?: string; // 搜索值
  block?: boolean;      // 是否撑满父容器宽度
}
