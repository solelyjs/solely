Drawer 抽屉

抽屉组件，从屏幕边缘滑出的浮层面板。

基础用法

点击按钮打开抽屉。

打开抽屉
等待交互...
import { Drawer } from 'solely/components/commands';

const drawer = Drawer.open({
    title: '抽屉标题',
    content: '抽屉内容',
    placement: 'right',
    onClose: () => {
        console.log('抽屉已关闭');
    }
});

drawer.close();

不同位置

抽屉可以从四个方向打开。

左侧抽屉 右侧抽屉 顶部抽屉 底部抽屉 
// 左侧抽屉
Drawer.open({
    placement: 'left',
    title: '左侧抽屉',
    content: '内容...',
    width: 250
});

// 右侧抽屉
Drawer.open({
    placement: 'right',
    title: '右侧抽屉',
    content: '内容...',
    width: 300
});

// 顶部抽屉
Drawer.open({
    placement: 'top',
    title: '顶部抽屉',
    content: '内容...',
    height: 200
});

// 底部抽屉
Drawer.open({
    placement: 'bottom',
    title: '底部抽屉',
    content: '内容...',
    height: 300
});

无遮罩层

设置 mask: false 可不显示遮罩层。

无遮罩抽屉 
Drawer.open({
    title: '无遮罩抽屉',
    content: '没有遮罩层的抽屉',
    placement: 'right',
    mask: false
});

不可关闭

设置 closable: false 隐藏关闭按钮，maskClosable: false 禁止点击遮罩关闭。

不可关闭抽屉 
Drawer.open({
    title: '不可关闭抽屉',
    content: '没有关闭按钮的抽屉',
    placement: 'right',
    closable: false,
    maskClosable: false,
    onClose: () => {
        console.log('抽屉已关闭');
    }
});

// 3秒后自动关闭
setTimeout(() => {
    Drawer.destroy();
}, 3000);

自定义样式

通过 bodyStyle 自定义抽屉内容区域的样式。

自定义抽屉 
Drawer.open({
    title: '自定义抽屉',
    content: '自定义内容和样式的抽屉',
    placement: 'right',
    width: 400,
    maskClosable: false,
    bodyStyle: { padding: '24px', backgroundColor: '#f5f5f5' },
    onClose: () => {
        console.log('抽屉已关闭');
    }
});

更新抽屉

使用 update() 方法更新抽屉的标题、内容、宽度/高度等属性。

更新抽屉 
const drawer = Drawer.open({
    title: '可更新的抽屉',
    content: '这是初始内容',
    width: 300
});

// 更新标题和内容
drawer.update({
    title: '已更新',
    content: '内容已更新'
});

// 更新宽度
drawer.update({ width: 500 });

// 更新样式
drawer.update({
    className: 'my-custom-class',
    bodyStyle: { backgroundColor: '#f0f0f0' }
});

全局配置

使用 config() 方法配置全局默认设置。

更新全局配置 销毁所有抽屉 
Drawer.config({
    width: 500,
    maskClosable: false
});

Drawer.destroy();

API

Drawer.open(options)

参数 说明 类型 默认值
title 抽屉标题 string -
content 抽屉内容 string -
placement 抽屉位置 'left' | 'right' | 'top' | 'bottom' 'right'
width 抽屉宽度（placement 为 left/right 时有效） number | string 378
height 抽屉高度（placement 为 top/bottom 时有效） number | string 378
mask 是否显示遮罩 boolean true
maskClosable 点击遮罩层是否关闭 boolean true
closable 是否显示关闭按钮 boolean true
className 自定义类名 string -
style 自定义样式 Partial<CSSStyleDeclaration> -
bodyStyle 抽屉内容的样式 Partial<CSSStyleDeclaration> -
onClose 关闭后回调 () => void -

DrawerInstance

方法 说明 参数
close() 关闭抽屉 -
update(options) 更新抽屉配置 options: Partial<DrawerOptions>

Drawer.config(options)

参数 说明 类型 默认值
width 默认宽度 number | string 378
height 默认高度 number | string 378
mask 默认是否显示遮罩 boolean true
maskClosable 默认点击遮罩是否关闭 boolean true
closable 默认是否显示关闭按钮 boolean true
