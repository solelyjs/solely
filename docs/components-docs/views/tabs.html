Tabs 标签页

用于内容的分组展示，支持切换、关闭、新增等操作。

基础用法

最简单的用法，通过 items 属性设置标签页数据。

内容 1
内容 2
内容 3
 
<solely-tabs
  items='[{"key":"1","label":"Tab 1"},{"key":"2","label":"Tab 2"},{"key":"3","label":"Tab 3"}]'
  activeKey="1">
  <div slot="tab-1">内容 1</div>
  <div slot="tab-2">内容 2</div>
  <div slot="tab-3">内容 3</div>
</solely-tabs>

可关闭

设置 closable 属性显示关闭按钮，点击会触发 close 事件。

标签 1 内容
标签 2 内容
标签 3 内容
 
<solely-tabs closable
  items='[{"key":"1","label":"标签 1"},{"key":"2","label":"标签 2"},{"key":"3","label":"标签 3"}]'
  activeKey="1"
  @close="handleClose(event)">
  <div slot="tab-1">标签 1 内容</div>
  <div slot="tab-2">标签 2 内容</div>
  <div slot="tab-3">标签 3 内容</div>
</solely-tabs>

可新增

设置 addable 属性显示新增按钮，点击会触发 add 事件。

首页内容
设置内容
 
<solely-tabs addable closable
  items='[{"key":"1","label":"首页"},{"key":"2","label":"设置"}]'
  activeKey="1"
  @add="handleAdd()">
  <div slot="tab-1">首页内容</div>
  <div slot="tab-2">设置内容</div>
</solely-tabs>

  @close="handleClose(event)"

<!-- JavaScript -->
// change 事件 - 通过 event.target.activeKey 获取当前激活的标签
const tabs = document.querySelector('solely-tabs');
tabs.addEventListener('change', (e) => {
  console.log('当前激活标签:', e.target.activeKey);
});

// close 事件 - 通过 event.detail 获取关闭的标签信息
tabs.addEventListener('close', (e) => {
  console.log('关闭标签:', e.detail.key, e.detail.label);
});

// add 事件
tabs.addEventListener('add', () => {
  console.log('新增标签');
});

API

属性

属性名 说明 类型 默认值
activeKey 当前激活标签页的 key string -
type 标签页类型，可选值为 line、card string line
position 标签页位置，可选值为 top、right、bottom、left string top
size 标签页大小，可选值为 small、default、large string default
items 标签页数据（JSON字符串） string []
closable 是否可关闭 boolean false
addable 是否可新增 boolean false

事件

事件名 说明 回调参数
change 切换标签时触发 Event
close 关闭标签时触发 CustomEvent<{ key: string; label: string }>
add 点击新增按钮时触发 Event

方法

方法名 说明 参数
setActiveKey(key) 设置当前激活的标签 key: string
getActiveKey() 获取当前激活的标签 -
setItems(items) 设置标签项 items: TabItem[]

类型定义

// 标签项类型
interface TabItem {
  key: string;      // 标签页标识
  label: string;    // 标签页标题
  disabled?: boolean; // 是否禁用
}

// Tabs 类型
type TabsType = 'line' | 'card';

// Tabs 位置
type TabsPosition = 'top' | 'right' | 'bottom' | 'left';

// Tabs 尺寸
type TabsSize = 'small' | 'default' | 'large';

插槽

插槽名 说明
tab-{key} 对应 key 的标签页内容，如 tab-1、tab-2
