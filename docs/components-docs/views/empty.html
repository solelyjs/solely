Empty 空状态

用于页面无数据时的占位展示，支持自定义图片和描述文字。

基础用法

最简单的空状态用法，使用默认图标和描述。

<solely-empty></solely-empty>

自定义描述

使用 description 属性或默认插槽自定义描述文字。

<solely-empty description="暂无数据"></solely-empty>
<solely-empty description="搜索无结果，请尝试其他关键词"></solely-empty>

自定义图片

使用 image 属性设置自定义图片 URL。

<solely-empty 
  image="https://example.com/image.png" 
  description="自定义图片">
</solely-empty>

图片插槽

使用 image 插槽可以使用更复杂的自定义图片内容，如 SVG 图标、动画图标等，优先级高于 image 属性。

📂
🔍

<solely-empty description="使用 Emoji 图标">
  <span slot="image" style="font-size: 64px;">📂</span>
</solely-empty>

<!-- 自定义样式容器 -->
<solely-empty description="自定义 SVG">
  <div slot="image" 
       style="width: 120px; height: 120px; background: linear-gradient(...); ...">
    <span style="font-size: 48px; color: #fff;">🔍</span>
  </div>
</solely-empty>

底部内容

使用默认插槽在底部添加操作按钮或其他内容。

创建数据 
<solely-empty description="暂无数据">
  <solely-button type="primary" size="small">创建数据</solely-button>
</solely-empty>

CSS Part 自定义样式

通过 ::part() 伪元素可在 Shadow DOM 外部自定义空状态内部样式，无需修改组件源码。

刷新页面 
<!-- HTML -->
<solely-empty description="暂无数据，请稍后重试">
  <solely-button type="primary" size="small">刷新页面</solely-button>
</solely-empty>

<!-- CSS：通过 ::part() 自定义样式 -->
<style>

    color: #667eea;

    font-weight: 600;
  }
  solely-empty::part(footer) {
    margin-top: 24px;
  }

API

属性

属性 说明 类型 默认值
description 描述文字 string ''
image 自定义图片 URL string ''

插槽

插槽名 说明
默认插槽 底部内容区域，常用于放置操作按钮
image 自定义图片内容，优先级高于 image 属性

CSS Part

Part 名称 说明 对应元素
base 空状态根容器 <div>
image 图片区域 <div>
img 自定义图片 <img>
icon 默认图标 <span>
description 描述文本 <p>
footer 底部操作区域 <div>

方法

方法名 说明 参数 返回值
setDescription(description) 设置描述文字 description: string void

类型定义

// 空状态属性接口
interface EmptyProps {
  description?: string;  // 描述文字
  image?: string;       // 图片 URL
}
