Empty 空状态

用于页面无数据时的占位展示，支持自定义图片和描述文字。

基础用法

最简单的空状态用法，使用默认图标和描述。

<solely-empty></solely-empty>

自定义描述

使用 description 属性或默认插槽自定义描述文字。

<solely-empty description="暂无数据"></solely-empty>
<solely-empty description="搜索无结果，请尝试其他关键词"></solely-empty>

自定义图片

使用 image 属性设置自定义图片 URL。

<solely-empty 
  image="https://example.com/image.png" 
  description="自定义图片">
</solely-empty>

底部内容

使用默认插槽在底部添加操作按钮或其他内容。

创建数据 
<solely-empty description="暂无数据">
  <solely-button type="primary" size="small">创建数据</solely-button>
</solely-empty>

API

属性

属性 说明 类型 默认值
description 描述文字 string ''
image 自定义图片 URL string ''

插槽

插槽名 说明
默认插槽 底部内容区域，常用于放置操作按钮

方法

方法名 说明 参数 返回值
setDescription(description) 设置描述文字 description: string void

类型定义

// 空状态属性接口
