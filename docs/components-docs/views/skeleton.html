Skeleton 骨架屏

在需要等待加载内容的位置提供一个占位图形组合，提升页面加载体验。

基础用法

最简单的骨架屏，显示段落占位。

内容已加载

这是真实的内容文本，加载完成后会显示在这里。

骨架屏会在加载时显示占位效果。
{{$data.loading ? '加载完成' : '重新加载'}} 
// HTML
<solely-skeleton id="skeleton">
  <div>
    <h3>内容已加载</h3>
    <p>这是真实的内容文本</p>
  </div>
</solely-skeleton>

// JavaScript
const skeleton = document.getElementById('skeleton');
skeleton.setAttribute('paragraph-rows', '3');
// 控制加载状态（布尔属性：存在即为 true）
skeleton.setAttribute('loading', '');      // 显示骨架屏
skeleton.removeAttribute('loading'); // 显示真实内容

带动画效果

使用 active 属性开启加载动画。

// HTML
<solely-skeleton id="skeleton"></solely-skeleton>

// JavaScript
const skeleton = document.getElementById('skeleton');
skeleton.setAttribute('active', '');      // 开启动画
skeleton.setAttribute('paragraph-rows', '3');

头像组合

使用 avatar 和 title 属性显示头像和标题占位。

// HTML
<solely-skeleton id="skeleton"></solely-skeleton>

// JavaScript
const skeleton = document.getElementById('skeleton');
skeleton.setAttribute('avatar', '');       // 显示头像
skeleton.setAttribute('title', '');        // 显示标题
skeleton.setAttribute('paragraph-rows', '2');

头像形状和大小

使用 avatar-shape 和 avatar-size 属性自定义头像。

<!-- 圆形头像（默认） -->
<solely-skeleton avatar title paragraph-rows="2"></solely-skeleton>

<!-- 方形头像 -->
<solely-skeleton avatar avatar-shape="square" title paragraph-rows="2"></solely-skeleton>

<!-- 大尺寸头像 -->
<solely-skeleton avatar avatar-size="64" title paragraph-rows="2"></solely-skeleton>

标题宽度

使用 title-width 属性设置标题宽度。

<solely-skeleton avatar title title-width="30%"></solely-skeleton>
<solely-skeleton avatar title title-width="50%"></solely-skeleton>
<solely-skeleton avatar title title-width="200px"></solely-skeleton>

段落行数

使用 paragraph-rows 属性设置段落行数。

<solely-skeleton title paragraph-rows="1"></solely-skeleton>
<solely-skeleton title paragraph-rows="3"></solely-skeleton>
<solely-skeleton title paragraph-rows="5"></solely-skeleton>

API

属性

属性 说明 类型 默认值
active 是否显示动画效果 boolean false
avatar 是否显示头像占位图 boolean false
avatar-shape 头像形状 'circle' | 'square' 'circle'
avatar-size 头像大小 number 40
title 是否显示标题占位图 boolean false
title-width 标题宽度 string | number ''
paragraph-rows 段落行数 number 3
loading 是否显示加载结束后的内容 boolean true

插槽

插槽名 说明
默认插槽 加载完成后显示的真实内容

方法

方法名 说明 参数 返回值
startLoading() 开始加载 - void
finishLoading() 结束加载 - void
setLoading(loading) 设置加载状态 loading: boolean void

类型定义

// 头像形状
type SkeletonAvatarShape = 'circle' | 'square';

// 骨架屏元素类型
type SkeletonElement = 'avatar' | 'title' | 'paragraph' | 'button' | 'input' | 'image';

// 骨架屏属性接口
interface SkeletonProps {
  active?: boolean;              // 是否显示动画
  avatar?: boolean;              // 是否显示头像
  avatarShape?: SkeletonAvatarShape; // 头像形状
  avatarSize?: number;            // 头像大小
  title?: boolean;               // 是否显示标题
  titleWidth?: string | number; // 标题宽度
  paragraphRows?: number;        // 段落行数
  loading?: boolean;           // 是否加载中
}
