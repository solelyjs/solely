FAB 悬浮按钮

悬浮在页面固定位置的按钮，常用于页面级别的全局操作。

基础用法

默认显示在右下角，点击可触发操作。

页面内容区域

点击事件

监听 click 事件来响应按钮点击，禁用状态下不会触发。

测试点击

  console.log('FAB 被点击了', event);

点击隐藏

使用 hide-on-click 属性，点击后自动隐藏按钮。

显示按钮

// 通过方法重新显示

API

属性

属性 说明 类型 默认值
type 按钮类型 'primary' | 'secondary' | 'success' | 'warning' | 'error' | 'default' 'primary'
size 按钮尺寸 'small' | 'medium' | 'large' 'medium'
shape 按钮形状 'circle' | 'square' | 'rounded' 'circle'
position 悬浮位置 'top-left' | 'top-center' | 'top-right' | 'left-center' | 'center' | 'right-center' | 'bottom-left' | 'bottom-center' | 'bottom-right' 'bottom-right'
offsetX 水平偏移量（像素） number 24
offsetY 垂直偏移量（像素） number 24
icon 图标内容 string '+'
absolute 是否使用绝对定位（用于演示） boolean false

事件

事件名 说明 回调参数
click 点击按钮时触发（禁用状态下不触发） MouseEvent

插槽

插槽名 说明
icon 自定义图标内容

类型定义

// FAB 按钮类型
type FabType = 
  | 'primary'
  | 'secondary'
  | 'success'
  | 'warning'
  | 'error'
  | 'default';

// FAB 按钮尺寸
type FabSize = 'small' | 'medium' | 'large';

// FAB 按钮形状
type FabShape = 'circle' | 'square' | 'rounded';

// FAB 位置
type FabPosition =
  | 'top-left'
  | 'top-center'
  | 'top-right'
  | 'left-center'
  | 'center'
  | 'right-center'
  | 'bottom-left'
  | 'bottom-center'
  | 'bottom-right';

// FAB 属性接口
interface FabProps {
  type?: FabType;        // 按钮类型
  size?: FabSize;        // 按钮尺寸
  shape?: FabShape;      // 按钮形状
  position?: FabPosition; // 悬浮位置
  offsetX?: number;      // 水平偏移量
  offsetY?: number;      // 垂直偏移量
  icon?: string;       // 图标内容
  absolute?: boolean;   // 是否绝对定位
}
