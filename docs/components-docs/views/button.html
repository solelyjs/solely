API

属性

属性 说明 类型 默认值
type 按钮类型 'primary' | 'secondary' | 'default' | 'dashed' | 'text' | 'link' | 'success' | 'warning' | 'error' 'default'
size 按钮尺寸 'small' | 'medium' | 'large' 'medium'
shape 按钮形状 'default' | 'round' | 'circle' | 'pill' 'default'
disabled 是否禁用 boolean false
loading 是否加载中 boolean false
ghost 幽灵按钮 boolean false
danger 危险按钮 boolean false
block 块级按钮 boolean false
icon 左侧图标 string -
icon-right 右侧图标 string -
icon-only 是否仅显示图标 boolean false
loading-icon 自定义加载图标 string -
href 链接地址，设置后渲染为 a 标签 string -
target 链接打开方式 string -

事件

事件名 说明 回调参数
click 点击按钮时触发（禁用或加载状态下不触发） MouseEvent

方法

方法名 说明
focus() 使按钮获得焦点
blur() 使按钮失去焦点

插槽

插槽名 说明
default 按钮内容
icon 左侧图标（优先级高于 icon 属性）
icon-right 右侧图标（优先级高于 icon-right 属性）

类型定义

// 按钮类型
type ButtonType = 
  | 'primary'
  | 'secondary'
  | 'default'
  | 'dashed'
  | 'text'
  | 'link'
  | 'success'
  | 'warning'
  | 'error';

// 按钮尺寸
type ButtonSize = 'small' | 'medium' | 'large';

// 按钮形状
type ButtonShape = 'default' | 'round' | 'circle' | 'pill';

// 按钮属性接口
interface ButtonProps {
  type?: ButtonType;      // 按钮类型
  size?: ButtonSize;      // 按钮尺寸
  shape?: ButtonShape;    // 按钮形状
  disabled?: boolean;      // 是否禁用
  loading?: boolean;       // 是否加载中
  ghost?: boolean;        // 幽灵按钮
  danger?: boolean;       // 危险按钮
  block?: boolean;        // 块级按钮
  icon?: string;          // 左侧图标
  iconRight?: string;     // 右侧图标
  iconOnly?: boolean;     // 仅图标
  loadingIcon?: string;   // 加载图标
  href?: string;          // 链接地址
  target?: string;        // 链接打开方式
}
