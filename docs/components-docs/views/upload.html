Upload 上传

文件上传组件，支持点击上传和拖拽上传。

基础用法

最简单的用法，点击上传文件。

<solely-upload></solely-upload>

自定义按钮文字

通过 button-text 属性自定义上传按钮文字。

<solely-upload button-text="上传图片"></solely-upload>
<solely-upload button-text="选择文件"></solely-upload>

多文件上传

设置 multiple 属性支持多文件上传。

<solely-upload multiple button-text="上传多个文件"></solely-upload>

限制文件类型

通过 accept 属性限制可上传的文件类型。

<solely-upload accept="image/*" button-text="只能上传图片"></solely-upload>
<solely-upload accept=".pdf,.doc,.docx" button-text="只能上传文档"></solely-upload>

拖拽上传

设置 drag 属性开启拖拽上传模式。

<solely-upload drag></solely-upload>

禁用状态

通过 disabled 属性禁用上传功能。

<solely-upload disabled></solely-upload>

限制数量

通过 max-count 属性限制最多上传文件数量。

<solely-upload multiple max-count="3" button-text="最多上传3个文件"></solely-upload>

限制大小

通过 max-size 属性限制单个文件大小（字节）。

<!-- 1MB = 1024 * 1024 = 1048576 字节 -->
<solely-upload max-size="1048576" button-text="单个文件不超过1MB"></solely-upload>

不同列表类型

通过 list-type 属性设置上传列表的样式。

<solely-upload list-type="text"></solely-upload>
<solely-upload list-type="picture"></solely-upload>

Block 属性

设置 block 属性可使上传按钮宽度撑满父容器。

<solely-upload block button-text="Block 上传按钮"></solely-upload>
<solely-upload block drag></solely-upload>

事件处理

监听 change、success、error、remove 事件处理上传逻辑。

事件日志（最近5条）：
等待交互...
<!-- HTML -->
<solely-upload 
  @change="handleChange(event)"
  @success="handleSuccess(event)"
  @error="handleError(event)"
  @remove="handleRemove(event)"
></solely-upload>

<!-- JavaScript -->
const uploadEl = document.querySelector('solely-upload');

// change 事件 - 文件列表变化时触发
uploadEl.addEventListener('change', (e) => {
  const fileList = e.target.fileList;
  console.log('文件列表:', fileList);
});

// success 事件 - 上传成功时触发
uploadEl.addEventListener('success', (e) => {
  console.log('上传成功:', e.detail.file);
});

// error 事件 - 上传失败时触发
uploadEl.addEventListener('error', (e) => {
  console.log('上传失败:', e.detail.message);
});

// remove 事件 - 移除文件时触发
uploadEl.addEventListener('remove', (e) => {
  console.log('移除文件:', e.detail.file);
});

API

属性

属性名 说明 类型 默认值
action 上传地址 string -
accept 接受上传的文件类型 string -
multiple 是否支持多选 boolean false
fileList 上传文件列表（JSON字符串） string []
listType 上传列表的内建样式（text/picture/picture-card） string text
disabled 是否禁用 boolean false
showUploadList 是否显示上传列表 boolean true
buttonText 上传按钮文字 string 上传文件
drag 是否支持拖拽上传 boolean false
maxCount 最大文件数量 number Infinity
maxSize 文件大小限制（字节） number Infinity
block 是否撑满父容器宽度 boolean false

事件

事件名 说明 回调参数
change 文件列表变化时触发 Event
success 上传成功时触发 CustomEvent<{ file: UploadFile; fileList: UploadFile[] }>
error 上传失败时触发 CustomEvent<{ file: UploadFile; message: string }>
remove 移除文件时触发 CustomEvent<{ file: UploadFile; fileList: UploadFile[] }>

方法

方法名 说明 参数 返回值
getFileList() 获取文件列表 - UploadFile[]
clear() 清空文件列表 - void

类型定义

// 上传文件类型
interface UploadFile {
  uid: string;           // 文件唯一标识
  name: string;          // 文件名
  size?: number;         // 文件大小
  type?: string;         // 文件类型
  status?: 'uploading' | 'done' | 'error' | 'removed';
  percent?: number;      // 上传进度
  url?: string;         // 文件URL
  error?: string;       // 错误信息
}

// 组件属性接口
interface UploadProps {
  action?: string;       // 上传地址
  accept?: string;       // 接受上传的文件类型
  multiple?: boolean;   // 是否支持多选
  fileList?: string;   // 上传文件列表（JSON字符串）
  listType?: 'text' | 'picture' | 'picture-card';
  disabled?: boolean;   // 是否禁用
  showUploadList?: boolean; // 是否显示上传列表
  buttonText?: string;  // 上传按钮文字
  drag?: boolean;       // 是否支持拖拽上传
  maxCount?: number;   // 最大文件数量
  maxSize?: number;    // 文件大小限制（字节）
  block?: boolean;      // 是否撑满父容器宽度
}
