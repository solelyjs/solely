Alert 警告提示

用于页面中展示重要的提示信息，适用于需要引起用户注意的场景。

基础用法

最简单的用法，适用于简短的提示。

<solely-alert type="info" message="Info text"></solely-alert>
<solely-alert type="success" message="Success text"></solely-alert>
<solely-alert type="warning" message="Warning text"></solely-alert>
<solely-alert type="error" message="Error text"></solely-alert>

可关闭

显示关闭按钮，点击可关闭警告提示。

<solely-alert type="info" message="Info Text" closable></solely-alert>
<solely-alert type="success" message="Success Text" closable></solely-alert>
<solely-alert type="warning" message="Warning Text" closable></solely-alert>
<solely-alert type="error" message="Error Text" closable></solely-alert>

含有辅助性文字介绍

含有辅助性文字介绍的警告提示。

<solely-alert type="info" message="Info Text"
  description="Detailed description..." show-icon></solely-alert>
<solely-alert type="success" message="Success Text"
  description="Detailed description..." show-icon></solely-alert>
<solely-alert type="warning" message="Warning Text"
  description="Detailed description..." show-icon></solely-alert>
<solely-alert type="error" message="Error Text"
  description="Detailed description..." show-icon></solely-alert>

事件监听

监听关闭事件。

<solely-alert type="info" message="Click the close button" closable @close="handleClose()"></solely-alert>

<script>
function handleClose() {
    console.log('Alert closed');
}
</script>

块级显示

添加 block 属性可以让警告提示宽度占满父容器。

<!-- 块级显示，宽度占满父容器 -->
<solely-alert block type="info" message="块级显示的 Info 警告提示"></solely-alert>
<solely-alert block type="success" message="块级显示的 Success 警告提示"></solely-alert>

API

属性

属性 说明 类型 默认值
type 指定警告提示的样式，有四种选择：info、success、warning、error string info
message 警告提示内容 string -
description 警告提示的辅助性文字介绍 string -
closable 是否显示关闭按钮 boolean false
showIcon 是否显示辅助图标 boolean false
block 是否块级显示 boolean false

事件

事件名 说明 回调参数
close 关闭时触发的事件 -

方法

方法名 说明 参数 返回值
setMessage 设置消息内容 message: string void
setType 设置类型 type: AlertType void

类型定义

type AlertType = 'info' | 'success' | 'warning' | 'error';

interface AlertProps {
    type?: AlertType;
    message?: string;
    description?: string;
    closable?: boolean;
    showIcon?: boolean;
    block?: boolean;
}
