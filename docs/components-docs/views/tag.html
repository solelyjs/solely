Tag 标签

用于标记和分类的标签组件，支持多种类型、尺寸和可关闭功能。

基础用法

使用 type 属性设置标签类型，默认为 default。

<solely-tag content="默认"></solely-tag>
<solely-tag type="primary" content="主要"></solely-tag>
<solely-tag type="success" content="成功"></solely-tag>
<solely-tag type="warning" content="警告"></solely-tag>
<solely-tag type="error" content="错误"></solely-tag>

尺寸

使用 size 属性设置标签尺寸，支持 small、medium（默认）、large。

<solely-tag type="primary" size="small" content="小尺寸"></solely-tag>
<solely-tag type="primary" size="medium" content="中尺寸"></solely-tag>
<solely-tag type="primary" size="large" content="大尺寸"></solely-tag>

圆角

使用 round 属性设置圆角标签。

<solely-tag type="primary" round content="圆角标签"></solely-tag>

可关闭

使用 closable 属性设置可关闭标签，点击关闭按钮会触发 close 事件并从 DOM 中移除。

已关闭: {{$data.closedTags.join(', ')}} 
<solely-tag type="primary" closable content="可关闭" @close="handleClose"></solely-tag>

禁用状态

使用 disabled 属性设置禁用状态，禁用后关闭按钮不可用。

<solely-tag type="primary" disabled content="禁用标签"></solely-tag>
<solely-tag type="primary" disabled closable content="禁用且可关闭"></solely-tag>

自定义颜色

使用 color 属性设置自定义背景色。

<solely-tag color="#722ed1" content="紫色"></solely-tag>
<solely-tag color="#eb2f96" content="粉色"></solely-tag>
<solely-tag color="#13c2c2" content="青色"></solely-tag>
<solely-tag color="#fa8c16" content="橙色"></solely-tag>

API

属性

属性 说明 类型 默认值
type 标签类型 'default' | 'primary' | 'success' | 'warning' | 'error' 'default'
size 标签尺寸 'small' | 'medium' | 'large' 'medium'
content 标签内容 string ''
closable 是否可关闭 boolean false
disabled 是否禁用 boolean false
round 是否为圆角 boolean false
color 自定义颜色 string ''

插槽

插槽名 说明
默认插槽 标签内容，优先级高于 content 属性

事件

事件名 说明 回调参数
close 关闭标签时触发 CustomEvent<{ content: string }>

方法

方法名 说明 参数 返回值
setContent(content) 设置标签内容 content: string void
setType(type) 设置标签类型 type: TagType void

类型定义

// 标签类型
type TagType = 'default' | 'primary' | 'success' | 'warning' | 'error';

// 标签尺寸
type TagSize = 'small' | 'medium' | 'large';

// 标签属性接口
interface TagProps {
  type?: TagType;        // 标签类型
  size?: TagSize;        // 标签尺寸
  content?: string;     // 标签内容
  closable?: boolean;   // 是否可关闭
  disabled?: boolean;   // 是否禁用
  round?: boolean;      // 是否为圆角
  color?: string;       // 自定义颜色
}
