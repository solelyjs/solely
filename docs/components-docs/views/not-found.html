NotFound 404 页面

用于页面未找到时的展示，支持自定义标题、描述和返回按钮。

基础用法

最简单的 404 页面用法，使用默认样式和内容。

<solely-not-found></solely-not-found>

<!-- 全屏模式（页面级使用） -->
<solely-not-found></solely-not-found>

<!-- 嵌入模式（文档/组件内使用） -->
<solely-not-found embedded></solely-not-found>

自定义内容

使用属性自定义标题、描述和按钮文字。

<solely-not-found 
  title="404" 
  subtitle="页面走丢了"
  description="您访问的页面可能已被删除或暂时无法访问"
  back-text="返回首页"
  back-url="/">
</solely-not-found>

CSS Part 自定义样式

通过 ::part() 伪元素可在 Shadow DOM 外部自定义 404 页面内部样式，无需修改组件源码。

<!-- HTML -->
<solely-not-found embedded></solely-not-found>

<!-- CSS：通过 ::part() 自定义样式 -->
<style>
  solely-not-found::part(base) {
    background: linear-gradient(135deg, #f0f5ff 0%, #e6f7ff 100%);
    border-radius: 12px;
  }
  solely-not-found::part(numbers) {
    color: #667eea;
    text-shadow: 0 4px 12px rgba(102, 126, 234, 0.3);
  }
  solely-not-found::part(icon-wrapper) {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);
  }
  solely-not-found::part(button) {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    border: none;
    border-radius: 8px;
    box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);
  }
  solely-not-found::part(button):hover {
    transform: translateY(-2px);
    box-shadow: 0 6px 16px rgba(102, 126, 234, 0.5);
  }
</style>

在路由中使用

配合 router-view 使用，作为 404 页面展示。

// 方式一：在 router-view 上指定
<router-view not-found="solely-not-found"></router-view>

// 方式二：在路由配置中指定
const routes: RouteConfig[] = [
  {
    path: '/',
    tagName: 'home-page',
    notFoundComponent: 'solely-not-found'
  }
];

API

属性

属性 说明 类型 默认值
title 主标题 string '404'
subtitle 副标题 string '抱歉，您访问的页面不存在'
description 描述文字 string '请检查网址是否正确，或点击下方按钮返回首页'
backText 返回按钮文字 string '返回首页'
backUrl 返回链接地址 string '/'
embedded 是否为嵌入模式（用于文档演示等场景） boolean false

CSS Part

Part 名称 说明 对应元素
base 404 页面根容器 <div>
container 内容容器 <div>
numbers 404 数字区域 <div>
number 单个数字 <span>
icon-wrapper 图标包装器 <div>
icon 感叹号图标 <svg>
title 主标题 <h1>
subtitle 副标题 <h2>
description 描述文本 <p>
button 返回按钮 <button>
button-icon 按钮内图标 <svg>
bg-decoration 装饰性背景元素 <div>

方法

方法名 说明 参数 返回值
handleBack() 处理返回按钮点击，优先使用路由跳转 - void

类型定义

// 404 组件属性接口
interface NotFoundProps {
  title?: string;        // 主标题
  subtitle?: string;     // 副标题
  description?: string;  // 描述文字
  backText?: string;     // 返回按钮文字
  backUrl?: string;      // 返回链接地址
  embedded?: boolean;    // 是否为嵌入模式
}

路由配置

// RouteConfig 扩展属性
interface RouteConfig {
  // ... 其他属性
  
  /** 404 页面组件标签名 */
  notFoundComponent?: string;
}
