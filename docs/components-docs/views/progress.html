API

属性

属性 说明 类型 默认值
type 进度条类型 'line' | 'circle' 'line'
percent 百分比（0-100） number 0
status 状态 'normal' | 'success' | 'exception' | 'active' 'normal'
showInfo 是否显示进度数值或状态图标 boolean true
strokeWidth 进度条线的宽度 number 8
width 圆形进度条画布宽度（仅在 type="circle" 时有效） number 120
format 内容的模板函数（仅在 type="line" 时有效） string ''

方法

方法名 说明 参数 返回值
setPercent(percent) 设置进度 percent: number void
getPercent() 获取进度（返回限制在 0-100 之间的值） - number

类型定义

// 进度条类型
type ProgressType = 'line' | 'circle';

// 进度条状态
type ProgressStatus = 'normal' | 'success' | 'exception' | 'active';

// 进度条属性接口
interface ProgressProps {
  type?: ProgressType;       // 进度条类型
  percent?: number;           // 百分比
  status?: ProgressStatus;   // 状态
  showInfo?: boolean;        // 是否显示进度数值
  strokeWidth?: number;     // 线条宽度
  width?: number;           // 圆形进度条宽度
  format?: string;         // 格式模板
}
