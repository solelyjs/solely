Divider 分割线

用于内容的分隔，支持水平/垂直方向、虚线样式和标题文字。

基础用法

最简单的分割线用法，默认为水平分割线。

这是一段文字

这是另一段文字

<p>这是一段文字</p>
<solely-divider></solely-divider>
<p>这是另一段文字</p>

带文字的分割线

使用 content 属性或默认插槽在分割线中间添加文字。

插槽文字 
<!-- 使用 content 属性 -->

<!-- 使用插槽 -->
<solely-divider>插槽文字</solely-divider>

使用 orientation 属性设置文字的位置，支持 left、right、center。

<solely-divider content="居中"></solely-divider>
<solely-divider content="右对齐" orientation="right"></solely-divider>

使用 dashed 属性设置虚线样式。

API

属性

属性 说明 类型 默认值
type 分割线类型 'horizontal' | 'vertical' 'horizontal'
orientation 文字位置（仅水平分割线有效） 'left' | 'right' | 'center' 'center'
dashed 是否虚线 boolean false
content 分割线文字内容 string ''

插槽

插槽名 说明
默认插槽 分割线文字内容，优先级高于 content 属性

类型定义

// 分割线类型
type DividerType = 'horizontal' | 'vertical';

// 文字位置
type DividerOrientation = 'left' | 'right' | 'center';

// 分割线属性接口
interface DividerProps {
  type?: DividerType;           // 分割线类型
  orientation?: DividerOrientation; // 文字位置
  dashed?: boolean;            // 是否虚线
  content?: string;           // 文字内容
}
