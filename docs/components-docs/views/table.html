Table 表格

表格组件，用于展示结构化数据。

基础用法

通过 columns 和 data-source 属性传入表格配置和数据。

<solely-table 
  columns='[{"title":"姓名","dataIndex":"name"},...]' 
  data-source='[{"name":"张三","age":28},...]'
></solely-table>

不同尺寸

支持 small、middle、default 三种尺寸。

<solely-table size="small" columns="..." data-source="..."></solely-table>
<solely-table size="middle" columns="..." data-source="..."></solely-table>
<solely-table columns="..." data-source="..."></solely-table>

带边框

设置 bordered 属性显示边框。

<solely-table bordered columns="..." data-source="..."></solely-table>

斑马纹

设置 striped 属性显示斑马纹。

<solely-table striped columns="..." data-source="..."></solely-table>

加载状态

设置 loading 属性显示加载状态。

<solely-table loading columns="..." data-source="[]"></solely-table>

空状态

当数据为空时显示空状态。

<solely-table columns="..." data-source="[]"></solely-table>

列对齐方式

通过 align 属性设置列的对齐方式（left/center/right）。

<solely-table 
  columns='[{"title":"居中","align":"center"},...]' 
  data-source="..."
></solely-table>

可点击行

设置 row-clickable 属性使表格行可点击，点击时触发 row-click 事件。

事件日志（最近5条）：
点击表格行查看事件...
<!-- HTML -->
<solely-table 
  row-clickable 
  @row-click="handleRowClick(event)"
  columns="..." 
  data-source="..."
></solely-table>

<!-- JavaScript -->
const tableEl = document.querySelector('solely-table');

tableEl.addEventListener('row-click', (e) => {
  const { row, index } = e.detail;
  console.log('点击了第', index + 1, '行:', row);
});

API

属性

属性名 说明 类型 默认值
columns 表格列配置（JSON字符串） string []
dataSource 表格数据（JSON字符串） string []
size 表格尺寸（small/middle/default） string default
bordered 是否显示边框 boolean false
loading 是否加载中 boolean false
striped 是否显示斑马纹 boolean false
rowClickable 是否可点击行 boolean false
rowKey 行数据的唯一标识字段名 string -

事件

事件名 说明 回调参数
click 点击行时触发（rowClickable 为 true 时） MouseEvent
row-click 点击行时触发的自定义事件 CustomEvent<{ row: Record<string, unknown>; index: number }>

方法

方法名 说明 参数 返回值
setColumns(columns) 设置列配置 columns: TableColumn[] void
setDataSource(dataSource) 设置数据源 dataSource: Record<string, unknown>[] void
getDataSource() 获取数据源 - Record<string, unknown>[]

类型定义

// 表格列配置
interface TableColumn {
  title: string;           // 列标题
  dataIndex: string;       // 列数据字段
  key: string;           // 列标识
  width?: number | string; // 列宽度
  fixed?: 'left' | 'right'; // 是否固定列
  align?: 'left' | 'center' | 'right'; // 对齐方式
  slot?: string;         // 自定义渲染（使用 slot 名称）
}

// 组件属性接口
interface TableProps {
  columns?: string;      // 表格列配置（JSON字符串）
  dataSource?: string;   // 表格数据（JSON字符串）
  size?: 'small' | 'middle' | 'default'; // 表格尺寸
  bordered?: boolean;    // 是否显示边框
  loading?: boolean;     // 是否加载中
  striped?: boolean;     // 是否显示斑马纹
  rowClickable?: boolean; // 是否可点击行
  rowKey?: string;       // 行数据的唯一标识字段名
}
