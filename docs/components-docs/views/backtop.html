BackTop 回到顶部

用于页面长距离滚动后快速返回顶部的按钮组件。组件会自动检测最近的滚动父元素。

基础用法

将组件放在滚动容器内，当滚动距离超过 400px 时，会在容器右下角显示回到顶部按钮。

<!-- 在指定容器中使用 -->

  <solely-backtop></solely-backtop>
</div>

事件演示

监听 click 和 backtop-finish 事件，通过 event.target 获取组件状态。

// 监听 click 事件（原生事件）
backtop.addEventListener('click', (event) => {
  const target = event.target;
  console.log('点击了按钮', target.visibilityHeight);
});

// 监听 backtop-finish 事件（自定义事件）
backtop.addEventListener('backtop-finish', (event) => {
  console.log('动画结束');
});

通过属性读取配置

可以通过 event.target.visibilityHeight 和 event.target.duration 读取当前配置，或直接设置属性。

<!-- HTML -->
<solely-backtop id="my-backtop" visibility-height="150" duration="600"></solely-backtop>

<!-- JavaScript -->
const backtop = document.querySelector('#my-backtop');

// 读取配置
console.log(backtop.visibilityHeight); // 150
console.log(backtop.duration);         // 600

// 设置配置
backtop.visibilityHeight = 200;
backtop.duration = 800;

API

属性

属性 说明 类型 默认值
visibilityHeight 滚动高度达到此参数值时才显示按钮（支持属性读写） number 400
duration 回到顶部的动画持续时间，单位毫秒（支持属性读写） number 450

事件

事件名 说明 回调参数
click 点击按钮时触发（原生事件） MouseEvent
backtop-finish 回到顶部动画结束时触发（自定义事件） CustomEvent

类型定义

// BackTop 属性接口
interface BackTopProps {
  visibilityHeight?: number;  // 显示按钮的滚动高度阈值
  duration?: number;          // 回到顶部动画持续时间（毫秒）
}

使用指南

BackTop 组件会自动检测最近的滚动父元素（overflow-y 为 auto 或 scroll 的元素），并在该容器右下角显示按钮。

使用要点
📦
容器要求 容器需要设置 overflow-y: auto 或 overflow-y: scroll 才能被检测到
📍
放置位置 组件本身应该放在可滚动容器内部
🎯
定位方式 按钮使用固定定位，会显示在容器右下角（相对于视口）
🔀
多实例支持 多个 BackTop 组件可以共存，各自显示在自己的容器内
完整示例
<!-- 基础用法 -->
<div class="scroll-container">
  <div class="content">
    长内容...
  </div>
  <solely-backtop></solely-backtop>
</div>

.scroll-container {

  overflow-y: auto;
