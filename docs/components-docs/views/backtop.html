BackTop 回到顶部

用于页面长距离滚动后快速返回顶部的按钮组件。组件会自动检测最近的滚动父元素。

基础用法

将组件放在滚动容器内，当滚动距离超过 400px 时，会在容器右下角显示回到顶部按钮。

👇 向下滚动查看效果...

继续向下滚动...

到底了，点击右下角按钮回到顶部

<!-- 在页面中使用 -->
<solely-backtop></solely-backtop>

<!-- 在指定容器中使用 -->
<div style="overflow-y: auto; height: 300px; position: relative;">
  <solely-backtop></solely-backtop>
</div>

自定义显示高度

通过 visibility-height 属性设置按钮显示时距离顶部的滚动距离。

👇 滚动 100px 后显示按钮（默认是 400px）...

继续滚动...

到底了

<!-- 滚动 100px 后显示 -->
<solely-backtop visibility-height="100"></solely-backtop>

<!-- 滚动 200px 后显示 -->
<solely-backtop visibility-height="200"></solely-backtop>

自定义滚动时间

通过 duration 属性设置滚动动画的持续时间（毫秒）。

👇 快速滚动回顶部（200ms）...

继续滚动...

到底了

<!-- 快速滚动（200ms） -->
<solely-backtop duration="200"></solely-backtop>

<!-- 慢速滚动（1000ms） -->
<solely-backtop duration="1000"></solely-backtop>

点击事件

监听 click 事件，在回到顶部后执行自定义操作。

👇 点击按钮查看事件...

继续滚动...

到底了

<!-- HTML -->
<solely-backtop @click="handleClick()"></solely-backtop>

<!-- JavaScript -->
const backtop = document.querySelector('solely-backtop');
backtop.addEventListener('click', () => {
  console.log('已回到顶部');
});

API

属性

属性名 说明 类型 默认值
visibilityHeight 滚动高度达到此参数值时才显示按钮 number 400
duration 回到顶部的动画持续时间（毫秒） number 450

事件

事件名 说明 回调参数
click 点击按钮时触发 Event

使用指南

BackTop 组件会自动检测最近的滚动父元素（overflow-y 为 auto 或 scroll 的元素），并在该容器右下角显示按钮。

💡 使用要点
📦
容器要求 容器需要设置 overflow-y: auto 或 overflow-y: scroll 才能被检测到
📍
放置位置 组件本身应该放在可滚动容器内部
🎯
定位方式 按钮使用固定定位，会显示在容器右下角（相对于视口）
🔀
多实例支持 多个 BackTop 组件可以共存，各自显示在自己的容器内
📝 完整示例
<!-- 基础用法 -->
<div class="scroll-container">
  <div class="content">
    长内容...
  </div>
  <solely-backtop></solely-backtop>
</div>

.scroll-container {
  height: 500px;
  overflow-y: auto;
  position: relative;
}
