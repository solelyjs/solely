Tooltip 文字提示

简单的文字提示气泡框，支持多种触发方式和位置。

基础用法

使用 bind() 方法将提示绑定到元素，支持不同的触发方式。

悬停触发 点击触发
等待交互...
import { Tooltip } from 'solely/components/commands';

// 悬停触发
const tooltip = Tooltip.bind(element, {
    content: '悬停提示内容',
    trigger: 'hover',
    placement: 'top'
});

// 点击触发
const tooltip = Tooltip.bind(element, {
    content: '点击提示内容',
    trigger: 'click',
    placement: 'bottom'
});

// 聚焦触发
const tooltip = Tooltip.bind(input, {
    content: '聚焦提示内容',
    trigger: 'focus',
    placement: 'right'
});

不同位置

提示可以显示在不同位置。

上方提示 下方提示 左侧提示 右侧提示 
// 上方提示
Tooltip.bind(element, {
    content: '上方提示',
    trigger: 'click',
    placement: 'top'
});

// 下方提示
Tooltip.bind(element, {
    content: '下方提示',
    trigger: 'click',
    placement: 'bottom'
});

// 左侧提示
Tooltip.bind(element, {
    content: '左侧提示',
    trigger: 'click',
    placement: 'left'
});

// 右侧提示
Tooltip.bind(element, {
    content: '右侧提示',
    trigger: 'click',
    placement: 'right'
});

自定义样式

可以自定义提示的背景颜色。

自定义颜色 
Tooltip.bind(element, {
    content: '自定义颜色提示',
    trigger: 'click',
    placement: 'top',
    color: '#f5222d'
});

手动控制

使用 defaultVisible 默认显示，或通过 visible 属性受控显示，也可以使用实例方法手动控制。

默认显示 受控模式 隐藏所有提示 
// defaultVisible - 非受控模式
const tooltip = Tooltip.bind(element, {
    content: '手动控制提示',
    trigger: 'click',
    placement: 'top',
    defaultVisible: true
});

// visible - 受控模式（需要使用 trigger: 'manual'）
const tooltip = Tooltip.bind(element, {
    content: '受控提示',
    trigger: 'manual',
    placement: 'top',
    visible: true
});

// 手动控制
tooltip.show();
tooltip.hide();
tooltip.destroy();

// 隐藏所有提示
Tooltip.hide();

临时提示

使用 show() 方法显示临时提示，自动消失。

显示临时提示 
Tooltip.show(element, {
    content: '自动消失的提示（3秒后）',
    placement: 'top',
    duration: 3000
});

与 Element 组件结合

在 Solely Element 组件中使用 Tooltip，支持通过 ref 绑定到 Element 组件。

绑定到 Element 显示临时提示 隐藏所有
等待交互...
// 使用 ref 获取 Element 组件并绑定 Tooltip
const btn = this.refs.elementTooltipBtn;

const tooltip = Tooltip.bind(btn, {
    content: 'Element 组件提示',
    placement: 'top',
    trigger: 'hover'
});

// 对 Element 组件显示临时提示
Tooltip.show(btn, {
    content: '临时提示',
    placement: 'top',
    duration: 2000
});

API

Tooltip.bind(target, options)

参数 说明 类型 默认值
target 目标元素 HTMLElement -
content 提示内容 string -
placement 气泡框位置 'top' | 'bottom' | 'left' | 'right' | 'topLeft' | 'topRight' | 'bottomLeft' | 'bottomRight' | 'leftTop' | 'leftBottom' | 'rightTop' | 'rightBottom' 'top'
trigger 触发方式 'hover' | 'click' | 'focus' 'hover'
defaultVisible 默认是否显示 boolean false
color 背景颜色 string -
className 自定义类名 string -
style 自定义样式 Partial<CSSStyleDeclaration> -
onVisibleChange 显示隐藏回调 (visible: boolean) => void -

Tooltip.show(target, options)

参数 说明 类型 默认值
target 目标元素 HTMLElement -
content 提示内容 string -
placement 气泡框位置 TooltipPlacement 'top'
duration 自动隐藏时间（毫秒） number 3000
color 背景颜色 string -
className 自定义类名 string -
style 自定义样式 Partial<CSSStyleDeclaration> -

Tooltip.hide()

方法 说明 参数
hide() 隐藏所有提示 -

TooltipInstance

方法 说明 参数
show() 显示提示 -
hide() 隐藏提示 -
destroy() 销毁提示（解绑事件） -

Tooltip.config(options)

参数 说明 类型 默认值
placement 默认位置 TooltipPlacement 'top'
trigger 默认触发方式 TooltipTrigger 'hover'
duration 默认显示时长（show 方法使用） number 3000

TooltipOptions

参数 说明 类型 默认值
content 提示内容 string -
placement 气泡框位置 TooltipPlacement 'top'
trigger 触发方式 TooltipTrigger 'hover'
visible 是否显示（受控模式） boolean -
defaultVisible 默认是否显示（非受控模式） boolean false
color 背景颜色 string -
className 自定义类名 string -
style 自定义样式 Partial<CSSStyleDeclaration> -
onVisibleChange 显示隐藏回调 (visible: boolean) => void -
