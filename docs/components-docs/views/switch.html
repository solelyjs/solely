Switch 开关

开关选择器，用于在两种状态之间切换。

基础用法

最简单的用法，通过 checked 属性控制开关状态。

<solely-switch></solely-switch>
<solely-switch checked></solely-switch>

尺寸

支持 small、medium（默认）、large 三种尺寸。

<solely-switch size="small"></solely-switch>
<solely-switch size="medium"></solely-switch>
<solely-switch size="large"></solely-switch>

颜色类型

支持 primary（默认）、success、warning、error 四种颜色类型。

<solely-switch checked></solely-switch>
<solely-switch type="success" checked></solely-switch>
<solely-switch type="warning" checked></solely-switch>
<solely-switch type="error" checked></solely-switch>

带文字

通过 checkedChildren 和 unCheckedChildren 属性设置开关状态的文字。

<solely-switch checked-children="开" un-checked-children="关"></solely-switch>
<solely-switch checked-children="是" un-checked-children="否" checked></solely-switch>
<solely-switch checked-children="ON" un-checked-children="OFF" size="large"></solely-switch>

带图标

使用 Unicode 字符或 Emoji 作为开关状态的图标。

<solely-switch checked-children="✓" un-checked-children="✕"></solely-switch>
<solely-switch checked-children="☀" un-checked-children="☾" checked></solely-switch>
<solely-switch checked-children="👍" un-checked-children="👎" size="large"></solely-switch>

禁用状态

通过 disabled 属性禁用开关，禁用后无法交互。

<solely-switch disabled></solely-switch>
<solely-switch disabled checked></solely-switch>

加载状态

通过 loading 属性设置加载状态，加载中无法切换。

<solely-switch loading></solely-switch>
<solely-switch loading checked></solely-switch>
<solely-switch loading type="success" checked></solely-switch>

事件处理

监听 change 事件获取开关状态变化。点击开关或按钮查看效果。

当前状态: {{$data.switchStatus}} 调用 toggle() 
<!-- HTML -->
<solely-switch ref="mySwitch" @change="handleChange(event)"></solely-switch>
<span>当前状态: {{status}}</span>
<solely-button @click="toggleSwitch()">调用 toggle()</solely-button>

<!-- JavaScript -->
const switchEl = document.getElementById('my-switch');
const statusEl = document.getElementById('status');
const toggleBtn = document.getElementById('toggle-btn');

// 监听 change 事件（原生事件）
switchEl.addEventListener('change', (e) => {
  const checked = e.target.checked;
  statusEl.textContent = `当前状态: ${checked ? '开启' : '关闭'}`;
});

// 通过方法触发事件
toggleBtn.addEventListener('click', () => {
  switchEl.toggle(); // 会触发 change 事件
});

键盘操作

支持键盘操作，按 Tab 聚焦后，按 Enter 或 Space 键可切换开关状态。

按 Tab 聚焦，Enter/Space 切换

API

属性

属性 说明 类型 默认值
checked 指定当前是否选中（支持属性和方法操作） boolean false
size 开关大小 'small' | 'medium' | 'large' 'medium'
type 开关颜色类型 'primary' | 'success' | 'warning' | 'error' 'primary'
disabled 是否禁用 boolean false
loading 是否加载中 boolean false
checkedChildren 选中时的内容（文字或图标） string ''
unCheckedChildren 非选中时的内容（文字或图标） string ''

事件

事件名 说明 回调参数
change 开关状态变化时触发 Event

方法

方法名 说明 参数 返回值
toggle() 切换开关状态（触发 change 事件） - void
setChecked(checked) 设置开关状态（触发 change 事件） checked: boolean void
getChecked() 获取当前开关状态 - boolean
focus() 聚焦到开关元素 - void
blur() 从开关元素移除焦点 - void

类型定义

// 开关尺寸
type SwitchSize = 'small' | 'medium' | 'large';

// 开关颜色类型
type SwitchType = 'primary' | 'success' | 'warning' | 'error';

// 开关属性接口
interface SwitchProps {
  checked?: boolean;         // 开关状态
  size?: SwitchSize;        // 开关尺寸
  type?: SwitchType;        // 颜色类型
  disabled?: boolean;       // 是否禁用
  loading?: boolean;        // 是否加载中
  checkedChildren?: string; // 选中时的内容
  unCheckedChildren?: string; // 非选中时的内容
}

// 开关变化事件数据
interface SwitchChangeEvent {
  checked: boolean;         // 当前开关状态
  originalEvent: Event;     // 原始事件
}
