复选框组件，用于在多个选项中进行多选。

事件处理

监听 change 事件获取复选框状态变化。点击复选框或按钮查看效果。

当前状态: {{$data.checkedStatus}} 调用 toggle() 
<!-- HTML -->
<solely-checkbox ref="myCheckbox" @change="handleChange"></solely-checkbox>
<span>当前状态: {{status}}</span>
<solely-button @click="toggleCheckbox">调用 toggle()</solely-button>

<!-- JavaScript -->
const checkboxEl = document.getElementById('my-checkbox');
const statusEl = document.getElementById('status');
const toggleBtn = document.getElementById('toggle-btn');

// 监听 change 事件（原生事件）
checkboxEl.addEventListener('change', () => {
  const checked = checkboxEl.checked;
  statusEl.textContent = `当前状态: ${checked ? '选中' : '未选中'}`;
});

// 通过方法触发事件
toggleBtn.addEventListener('click', () => {
  checkboxEl.toggle(); // 会触发 change 事件
});

API

属性

属性 说明 类型 默认值
checked 指定当前是否选中（支持属性和方法操作） boolean false
indeterminate 指定当前是否为半选中状态 boolean false
disabled 是否禁用 boolean false
content 复选框文本内容 string ''
type 颜色类型 'primary' | 'success' | 'warning' | 'error' 'primary'
size 尺寸大小 'small' | 'medium' | 'large' 'medium'
block 是否块级显示 boolean false
name 表单 name 属性 string ''

事件

事件名 说明 回调参数
change 选中状态变化时触发（原生事件） Event

方法

方法名 说明 参数 返回值
toggle() 切换选中状态（触发 change 事件） - void
setChecked(checked) 设置选中状态 checked: boolean void
getChecked() 获取当前选中状态 - boolean
setIndeterminate(indeterminate) 设置半选状态 indeterminate: boolean void
focus() 聚焦到复选框 - void
blur() 从复选框移除焦点 - void

类型定义

// 复选框尺寸类型
type CheckboxSize = 'small' | 'medium' | 'large';

// 复选框颜色类型
type CheckboxType = 'primary' | 'success' | 'warning' | 'error';

// 复选框属性接口
interface CheckboxProps {
  checked?: boolean;       // 是否选中
  indeterminate?: boolean; // 是否半选
  disabled?: boolean;      // 是否禁用
  content?: string;       // 文本内容
  type?: CheckboxType;     // 颜色类型
  size?: CheckboxSize;     // 尺寸大小
  block?: boolean;        // 是否块级显示
  name?: string;         // 表单 name 属性
}
