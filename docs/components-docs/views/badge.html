API

属性

属性 说明 类型 默认值
count 显示的数字，大于 overflowCount 时显示为 ${overflowCount}+ number 0
dot 不展示数字，只有一个小红点 boolean false
overflowCount 展示封顶的数字值 number 99
status 设置状态点的颜色，可选：success、processing、default、error、warning string -
text 在设置了 status 的前提下有效，设置状态点的文本 string -
color 自定义背景颜色 string -

方法

方法名 说明 参数 返回值
setCount 设置数字 count: number void
increase 增加数字 delta?: number void
decrease 减少数字 delta?: number void
showDot 显示小红点 - void
hideDot 隐藏小红点 - void

类型定义

type BadgeStatus = 'success' | 'processing' | 'default' | 'error' | 'warning';

interface BadgeProps {
    count?: number;
    dot?: boolean;
    overflowCount?: number;
    status?: BadgeStatus;
    text?: string;
    color?: string;
}
