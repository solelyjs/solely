Message 全局提示

全局展示操作反馈信息，支持多种消息类型和自定义配置。

基础用法

提供五种不同级别的消息提示。

信息提示 成功提示 警告提示 错误提示
等待交互...
import { Message } from 'solely/components/commands';

Message.info('这是一条信息提示');
Message.success('操作成功完成');
Message.warning('请注意操作安全');
Message.error('操作失败，请重试');
Message.loading('正在加载中...');

加载状态

显示加载中的提示信息，默认不会自动关闭，需手动调用 close()。

显示加载中 隐藏加载中 
const loading = Message.loading('正在加载中...');

// 完成后手动关闭
loading.close();

// 也可以指定自动关闭时间
Message.loading('加载中', 5000);

自定义时长

可以自定义消息的显示时长，或设置 duration: 0 不自动关闭。

5秒后关闭 不自动关闭 
Message.success('这条提示将持续5秒', 5000);

Message.open({
    content: '这条提示不会自动关闭',
    type: 'info',
    duration: 0,
    closable: true
});

带描述的消息

消息可以包含标题和描述信息。

带描述的消息 
Message.open({
    content: '操作成功',
    description: '您的数据已成功保存到服务器',
    type: 'success',
    duration: 5000
});

自定义图标

可以传入自定义图标元素。

自定义图标 
const customIcon = document.createElement('span');
customIcon.textContent = '★';
customIcon.style.fontSize = '16px';

Message.open({
    content: '自定义图标消息',
    icon: customIcon,
    duration: 3000
});

更新消息内容

使用 update() 方法更新消息内容。

更新消息 
const instance = Message.loading('正在加载中...');

// 更新内容
instance.update('加载完成！');

// 或同时更新内容和描述
instance.update({
    content: '上传成功',
    description: '文件已保存'
});

全局配置

使用 config() 方法配置全局默认设置。

更新全局配置 销毁所有消息 
Message.config({
    duration: 5000,
    maxCount: 3,
    top: 50
});

Message.destroy();

API

Message 方法

方法 说明 参数 返回值
Message.open 打开消息 options: MessageOptions MessageInstance
Message.info 信息提示 content: string, duration?: number MessageInstance
Message.success 成功提示 content: string, duration?: number MessageInstance
Message.warning 警告提示 content: string, duration?: number MessageInstance
Message.error 错误提示 content: string, duration?: number MessageInstance
Message.loading 加载中提示（默认不自动关闭） content: string, duration?: number MessageInstance
Message.config 全局配置 options: Partial<MessageConfig> void
Message.destroy 销毁所有消息 - void

MessageOptions

参数 说明 类型 默认值
content 消息内容 string -
description 消息描述 string -
type 消息类型 'info' | 'success' | 'warning' | 'error' | 'loading' 'info'
showIcon 是否显示图标 boolean true
icon 自定义图标 string | HTMLElement -
duration 显示时长（毫秒），0 表示不自动关闭 number 3000
closable 是否显示关闭按钮 boolean false
className 自定义类名 string -
style 自定义样式 Partial<CSSStyleDeclaration> -
onClose 关闭时的回调 () => void -

MessageInstance

方法 说明 参数
close() 关闭当前消息 -
update(content) 更新消息内容或描述 string 或 { content?: string, description?: string }

MessageConfig

参数 说明 类型 默认值
duration 默认显示时长 number 3000
maxCount 最大显示数量 number 5
gap 消息之间的间距 number 8
top 距离顶部的距离 number 24
