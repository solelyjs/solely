Solely UI

轻量级 TypeScript Web Components 组件库

🚀 高性能 📦 轻量级 🔒 TypeScript 🎨 可定制

📖 介绍

Solely UI 是一个基于 TypeScript 的轻量级 Web Components 组件库，提供了丰富的 UI 组件和命令式组件， 帮助开发者快速构建现代化的 Web 应用。所有组件都基于 Web Components 标准构建，可以在任何框架中使用。

💡
框架无关

基于原生 Web Components 标准，可以在 React、Vue、Angular 等任何框架中使用，甚至可以与 jQuery 等库共存。

🚀 快速开始

在项目中引入 Solely UI 组件。

Primary Button Secondary Default 
// 引入主题 CSS（必需）
import 'solely/components/theme/variables.css';

// 引入单个组件
import 'solely/components/button';

// 或者引入所有组件
import 'solely/components';

// 在 HTML 中使用
<solely-button type="primary">Primary Button</solely-button>
<solely-input placeholder="请输入内容"></solely-input>

⚡ 命令式组件

Solely UI 提供了一系列命令式组件，可以通过 JavaScript 直接调用，无需在模板中声明。 这些组件非常适合用于全局提示、确认对话框等场景。

显示消息 显示对话框 显示抽屉 
import { Message, Modal, Drawer } from 'solely/components/commands';

// Message 全局提示
Message.success('操作成功！');
Message.error('操作失败！');
Message.loading('加载中...');

// Modal 对话框
Modal.confirm({
    title: '确认删除',
    content: '确定要删除此项吗？',
    onOk: () => {
        console.log('用户确认');
    }
});

// Drawer 抽屉
Drawer.open({
    title: '抽屉标题',
    content: '抽屉内容',
    placement: 'right'
});

🎨 主题定制

Solely UI 使用 CSS 变量定义主题，你可以轻松自定义主题颜色、字体、间距等。 

/* 自定义主题变量 */
:root {
  --solely-primary: #1890ff;
  --solely-success: #52c41a;
  --solely-warning: #faad14;
  --solely-error: #f5222d;
  --solely-font-size-md: 14px;
  --solely-border-radius-md: 6px;
}
