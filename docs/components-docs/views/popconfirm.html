Popconfirm 气泡确认框

点击元素，弹出气泡确认框，用于轻量级确认操作。

基础用法

基本的确认框使用方式。

删除项目
{{$data.popconfirmLog.length > 0 ? $data.popconfirmLog.join('\n') : '等待交互...'}}
import { Popconfirm } from 'solely/components/commands';

Popconfirm.show(target, {
    title: '确认删除',
    description: '您确定要删除此项吗？',
    onConfirm: () => {
        console.log('用户确认删除');
    },
    onCancel: () => {
        console.log('用户取消删除');
    }
});

自定义内容

可以自定义确认框的内容和按钮文字。

自定义确认框 危险操作确认 无取消按钮 
Popconfirm.show(target, {
    title: '确认操作',
    description: '此操作不可逆，请谨慎操作',
    okText: '我确定',
    cancelText: '再想想',
    onConfirm: () => {
        console.log('用户确认操作');
    }
});

Popconfirm.show(target, {
    title: '危险操作',
    description: '此操作具有风险，是否继续？',
    okText: '继续',
    cancelText: '取消',
    okType: 'danger',
    onConfirm: () => {
        console.log('用户确认危险操作');
    }
});

Popconfirm.show(target, {
    title: '仅确认操作',
    description: '这个确认框没有取消按钮',
    showCancel: false,
    onConfirm: () => {
        console.log('用户确认操作');
    }
});

绑定到元素

使用 bind() 方法将确认框绑定到元素，点击显示，点击外部关闭。

绑定确认框 
const instance = Popconfirm.bind(target, {
    title: '绑定确认框',
    description: '点击按钮显示，点击外部关闭',
    onConfirm: () => {
        console.log('用户确认');
    },
    onCancel: () => {
        console.log('用户取消');
    }
});

instance.destroy();

与 Element 组件结合

在 Solely Element 组件中使用 Popconfirm，支持通过 event.target 获取触发元素。

删除按钮 操作按钮 自定义 DOM 无图标
{{$data.elementPopconfirmLog.length > 0 ? $data.elementPopconfirmLog.join('\n') : '等待交互...'}}
// 在 Element 组件点击事件中触发 Popconfirm
const handleDelete = (event) => {
    const target = event.target;

    Popconfirm.show(target, {
        title: '确认删除',
        description: '此操作不可恢复',
        okType: 'danger',
        placement: 'top',
        onConfirm: () => {
            Message.success('删除成功');
        }
    });
};

// 使用 bind 绑定到 Element 组件
const btn = this.refs.actionBtn;
Popconfirm.bind(btn, {
    title: '确认操作',
    onConfirm: () => {}
});

// 使用 HTMLElement 作为标题和描述
const customTitle = document.createElement('div');
customTitle.innerHTML = '<span style="color: var(--solely-error);">⚠️ 警告</span>';

const customDesc = document.createElement('div');
customDesc.innerHTML = `此操作将 <strong>永久删除</strong> 数据，请确认`;

Popconfirm.show(btn, {
    title: customTitle,
    description: customDesc,
    okType: 'danger',
    placement: 'top'
});

// 去掉默认图标，完全自定义内容
Popconfirm.show(btn, {
    title: '确认操作',
    description: '是否继续？',
    showIcon: false,  // 去掉默认图标
    placement: 'top'
});

API

Popconfirm.show(target, options)

参数 说明 类型 默认值
target 触发元素 HTMLElement -
title 确认框标题 string | HTMLElement -
description 确认框描述 string | HTMLElement -
placement 气泡框位置 'top' | 'bottom' | 'left' | 'right' | 'topLeft' | 'topRight' | 'bottomLeft' | 'bottomRight' | 'leftTop' | 'leftBottom' | 'rightTop' | 'rightBottom' 'top'
okText 确定按钮文字 string | HTMLElement '确定'
okType 确定按钮类型 'primary' | 'danger' | 'default' 'primary'
cancelText 取消按钮文字 string | HTMLElement '取消'
showCancel 是否显示取消按钮 boolean true
showIcon 是否显示图标 boolean true
className 自定义类名 string -
style 自定义样式 Partial<CSSStyleDeclaration> -
onConfirm 确认回调函数 () => void -
onCancel 取消回调函数 () => void -

Popconfirm.bind(target, options)

参数 说明 类型 默认值
target 绑定元素 HTMLElement -
options 配置选项（同 show 方法） PopconfirmOptions -

PopconfirmInstance

方法 说明 参数
destroy() 销毁确认框 -

Popconfirm.config(options)

参数 说明 类型 默认值
okText 默认确认按钮文字 string | HTMLElement '确定'
cancelText 默认取消按钮文字 string | HTMLElement '取消'
okType 默认确认按钮类型 'primary' | 'danger' | 'default' 'primary'
showCancel 默认是否显示取消按钮 boolean true
placement 默认气泡框位置 'top' | 'bottom' | 'left' | 'right' 'top'

Popconfirm.destroy()

方法 说明 参数
destroy() 关闭最顶层的确认框 -
