Select 选择器

下拉选择器组件，用于从多个选项中选择一项。

基础用法

最简单的用法，通过 options 属性传入选项列表（JSON 字符串格式）。

<solely-select options='[{"label":"选项1","value":"1"},{"label":"选项2","value":"2"}]'></solely-select>

不同尺寸

通过 size 属性设置选择器尺寸，可选 small、medium（默认）、large。

<solely-select size="small" options='[...]'></solely-select>
<solely-select size="medium" options='[...]'></solely-select>
<solely-select size="large" options='[...]'></solely-select>

可清除

通过 clearable 属性设置是否显示清除按钮。

<solely-select clearable options='[...]'></solely-select>

禁用状态

通过 disabled 属性禁用选择器。

<solely-select disabled options='[...]'></solely-select>

事件处理

监听 change 事件获取选中的值，监听 clear 事件处理清除操作。通过 event.target.value 获取当前选中的值。

选中: {{$data.selectedLabel || '无'}} 
<!-- HTML -->
<solely-select 
  options='[...]' 
  @change="handleChange(event)"
  @clear="handleClear()">
</solely-select>

<!-- JavaScript -->
const select = document.querySelector('solely-select');

select.addEventListener('change', (e) => {
  // 通过 event.target.value 获取选中的值
  const value = e.target.value;
  console.log(`选中值: ${value}`);
});

select.addEventListener('clear', (e) => {
  console.log('已清除');
});

块级显示

添加 block 属性可以让选择器宽度占满父容器。

<!-- 块级显示，宽度占满父容器 -->
<solely-select 
  block
  options='[...]'>
</solely-select>

<solely-select 
  block
  size="large"
  options='[...]'>
</solely-select>

API

属性

属性 说明 类型 默认值
value 选中的值（支持属性和方法操作） string ''
placeholder 占位符文本 string '请选择'
disabled 是否禁用 boolean false
size 尺寸，可选 small、medium、large string 'medium'
clearable 是否显示清除按钮 boolean false
options 选项列表（JSON 字符串格式） string '[]'
multiple 是否多选（预留） boolean false
block 是否块级显示，设置为 true 时宽度占满父容器 boolean false

事件

事件名 说明 回调参数
change 选中值变化时触发，通过 event.target.value 获取值 Event
clear 点击清除按钮时触发 Event

方法

方法名 说明 参数 返回值
setValue(value) 设置选中的值 value: string void
getValue() 获取选中的值 - string
setOptions(options) 设置选项列表 options: SelectOption[] void

类型定义

// 选择器尺寸类型
type SelectSize = 'small' | 'medium' | 'large';

// 选项数据
interface SelectOption {
  label: string;      // 选项标签
  value: string;      // 选项值
  disabled?: boolean; // 是否禁用
}

// 选择器属性接口
interface SelectProps {
  value?: string;        // 选中的值
  placeholder?: string;  // 占位符
  disabled?: boolean;   // 是否禁用
  size?: SelectSize;   // 尺寸
  clearable?: boolean;  // 是否可清除
  options?: string;    // 选项列表（JSON 字符串）
  multiple?: boolean;  // 是否多选
  block?: boolean;    // 是否块级显示
}
