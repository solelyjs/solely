Select 选择器

下拉选择器组件，用于从多个选项中选择一项。支持自动调整下拉框位置、图标选项等功能。

基础用法

最简单的用法，通过 options 属性传入选项列表（JSON 字符串格式）。

<solely-select options='[{"label":"选项1","value":"1"},{"label":"选项2","value":"2"}]'></solely-select>

不同尺寸

通过 size 属性设置选择器尺寸，可选 small、medium（默认）、large。

<solely-select size="small" options='[...]'></solely-select>
<solely-select size="medium" options='[...]'></solely-select>
<solely-select size="large" options='[...]'></solely-select>

可清除

通过 clearable 属性设置是否显示清除按钮。

<solely-select clearable options='[...]'></solely-select>

禁用状态

通过 disabled 属性禁用选择器。

<solely-select disabled options='[...]'></solely-select>

禁用选项

可以在选项中设置 disabled 属性来禁用单个选项。

<solely-select options='[{
  "label": "可用选项",
  "value": "1"
}, {
  "label": "禁用选项",
  "value": "2",
  "disabled": true
}, ...]'>
</solely-select>

事件处理

监听 change 事件获取选中的值，监听 clear 事件处理清除操作。通过 event.target.value 获取当前选中的值。

选中: {{$data.selectedLabel || '无'}} 
<!-- HTML -->
<solely-select 
  options='[...]' 
  @change="handleChange(event)"
  @clear="handleClear()">
</solely-select>

<!-- JavaScript -->
const select = document.querySelector('solely-select');

select.addEventListener('change', (e) => {
  // 通过 event.target.value 获取选中的值
  const value = e.target.value;
  console.log(`选中值: ${value}`);
});

select.addEventListener('clear', (e) => {
  console.log('已清除');
});

块级显示

添加 block 属性可以让选择器宽度占满父容器。

<!-- 块级显示，宽度占满父容器 -->
<solely-select 
  block
  options='[...]'>
</solely-select>

<solely-select 
  block
  size="large"
  options='[...]'>
</solely-select>

自动调整位置

下拉框会根据视口空间自动决定向上或向下展开。当选择器位于页面底部且下方空间不足时，下拉框会自动向上展开。

💡 提示：点击选择器，观察下拉框向上展开（上方预留了足够空间） 
<!-- 自动调整位置 - 无需额外配置 -->
<!-- 当下方空间不足时，下拉框会自动向上展开 -->
<solely-select

</solely-select>

通过插槽可以完全自定义选项内容，支持任意 HTML 结构、图标、图片等。

双向绑定

使用 s-model 指令实现双向数据绑定，绑定的是 value 属性。

选中值: {{$data.modelSelect1 || '无'}}
可清除，选中值: {{$data.modelSelect2 || '无'}}
设置预设值 
<!-- 使用 s-model 实现双向绑定（绑定 value 属性） -->
<solely-select 
  s-model="selectedCity"
  options='[{"label":"北京","value":"beijing"}, ...]'>
</solely-select>

// s-model 会自动绑定 value 属性和监听 change 事件

API

属性

属性 说明 类型 默认值
value 选中的值（支持属性和方法操作） string ''
placeholder 占位符文本 string '请选择'
disabled 是否禁用 boolean false
size 尺寸，可选 small、medium、large string 'medium'
clearable 是否显示清除按钮 boolean false
options 选项列表（JSON 字符串格式） string '[]'
multiple 是否多选（预留） boolean false
block 是否块级显示，设置为 true 时宽度占满父容器 boolean false

事件

事件名 说明 回调参数
change 选中值变化时触发，通过 event.target.value 获取值 Event
clear 点击清除按钮时触发 Event

方法

方法名 说明 参数 返回值
setValue(value) 设置选中的值 value: string void
getValue() 获取选中的值 - string
setOptions(options) 设置选项列表 options: SelectOption[] void

类型定义

// 选择器尺寸类型
type SelectSize = 'small' | 'medium' | 'large';

// 图标类型 - 安全的使用方式
type SelectIcon = 
  | string                                          // CSS 类名（如字体图标 'icon-user'）
  | HTMLElement                                     // DOM 元素（推荐用于 SVG、图片等）
  | (option: SelectOption) => string | HTMLElement;  // 函数返回图标

// 选项数据
interface SelectOption {
  label: string;                    // 选项标签
  value: string;                    // 选项值
  disabled?: boolean;               // 是否禁用
  icon?: SelectIcon;               // 选项图标（string 仅作为 CSS 类名）
  render?: (option: SelectOption) => string | HTMLElement;  // 完全自定义渲染
}

// 选择器属性接口
interface SelectProps {
  value?: string;        // 选中的值
  placeholder?: string;  // 占位符
  disabled?: boolean;   // 是否禁用
  size?: SelectSize;   // 尺寸
  clearable?: boolean;  // 是否可清除
  options?: SelectOption[]; // 选项列表
  multiple?: boolean;  // 是否多选
  block?: boolean;    // 是否块级显示
}

方法

方法名 说明 参数 返回值
setValue(value) 设置选中的值 value: string void
getValue() 获取选中的值 - string
setOptions(options) 设置选项列表 options: SelectOption[] void
getOptions() 获取选项列表 - SelectOption[]
open() 打开下拉菜单 - void
close() 关闭下拉菜单 - void
clear() 清空选中的值 - void

类型定义

// 选择器尺寸类型

// 选项数据
interface SelectOption {
  label: string;                    // 选项标签
  value: string;                    // 选项值
  disabled?: boolean;               // 是否禁用
}

插槽使用说明

当需要在选项中使用复杂的 HTML 结构（如图标、图片、多行文本等）时，可以使用插槽方式：

<!-- 插槽方式：在 Select 内直接放置选项元素 -->
<solely-select>
  <!-- 每个选项需要 data-value 属性 -->
  <div data-value="value1" data-label="显示文本">
    <!-- 任意自定义内容 -->
    <img src="icon.png">
    <span>选项文本</span>
  </div>
  
  <!-- 禁用选项添加 disabled 属性 -->
  <div data-value="value2" disabled>...</div>
</solely-select>

插槽选项属性：

  • data-value（必需）：选项的值
  • data-label（可选）：选中后显示的标签，默认使用元素文本内容
  • disabled（可选）：禁用该选项
