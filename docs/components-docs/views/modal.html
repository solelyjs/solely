Modal 对话框

模态对话框，支持多种类型和自定义配置。

基础用法

基本的对话框使用方式。

打开对话框
{{$data.modalLog.length > 0 ? $data.modalLog.join('\n') : '等待交互...'}}
import { Modal } from 'solely/components/commands';

const modal = Modal.open({
    title: '对话框标题',
    content: '对话框内容',
    onConfirm: () => {
        console.log('用户点击了确定');
    },
    onCancel: () => {
        console.log('用户点击了取消');
    }
});

确认对话框

用于确认重要操作的对话框。

确认对话框 删除确认 
Modal.confirm({
    title: '确认操作',
    content: '您确定要执行此操作吗？',
    onConfirm: () => {
        console.log('用户确认了操作');
    },
    onCancel: () => {
        console.log('用户取消了操作');
    }
});

Modal.confirm({
    title: '删除确认',
    content: '此操作将永久删除数据，是否继续？',
    okText: '删除',
    cancelText: '取消',
    okType: 'danger',
    onConfirm: () => {
        console.log('用户确认删除');
    }
});

信息对话框

用于展示信息的对话框。

信息对话框 成功对话框 警告对话框 错误对话框 
Modal.info({
    title: '信息提示',
    content: '这是一条信息提示',
    onConfirm: () => {
        console.log('用户确认了信息');
    }
});

Modal.success({
    title: '操作成功',
    content: '您的操作已成功完成',
    onConfirm: () => {
        console.log('用户确认了成功信息');
    }
});

Modal.warning({
    title: '警告提示',
    content: '请注意操作风险',
    onConfirm: () => {
        console.log('用户确认了警告信息');
    }
});

Modal.error({
    title: '操作失败',
    content: '操作过程中出现错误',
    onConfirm: () => {
        console.log('用户确认了错误信息');
    }
});

异步操作

onConfirm 支持异步函数，操作完成后才会关闭对话框。

异步操作 
Modal.open({
    title: '异步操作对话框',
    content: '点击确定将执行异步操作',
    okText: '提交',
    onConfirm: async () => {
        await new Promise(resolve => setTimeout(resolve, 1500));
        console.log('异步操作完成');
    }
});

自定义按钮

通过 buttons 数组自定义底部按钮。

自定义按钮 
Modal.open({
    title: '自定义按钮对话框',
    content: '这个对话框有三个自定义按钮',
    buttons: [
        {
            text: '取消',
            type: 'default',
            onClick: () => {
                console.log('点击了取消');
            }
        },
        {
            text: '稍后处理',
            type: 'default',
            onClick: () => {
                console.log('点击了稍后处理');
            }
        },
        {
            text: '立即处理',
            type: 'primary',
            onClick: () => {
                console.log('点击了立即处理');
            }
        }
    ]
});

可点击遮罩关闭

设置 maskClosable: true 允许点击遮罩层关闭对话框。

点击遮罩关闭 
Modal.open({
    title: '可点击遮罩关闭',
    content: '点击遮罩层可以关闭此对话框',
    maskClosable: true,
    onConfirm: () => {
        console.log('用户点击了确定');
    },
    onCancel: () => {
        console.log('用户点击了遮罩或取消');
    }
});

更新对话框

使用 update() 方法动态更新对话框的标题、内容、宽度、按钮文字等属性。

更新对话框 
const modal = Modal.open({
    title: '可更新的对话框',
    content: '这是初始内容',
    width: 400,
    okText: '更新内容',
    onConfirm: () => {
        modal.update({
            title: '已更新',
            content: '内容已更新为新的文本',
            width: 600,
            className: 'my-updated-modal',
            style: { backgroundColor: '#f5f5f5' }
        });
    }
});

全局配置

使用 config() 方法配置全局默认设置。

更新全局配置 销毁所有对话框 
Modal.config({
    width: 600,
    okText: '确认',
    cancelText: '关闭'
});

Modal.destroy();

与 Element 组件结合

在 Solely Element 组件中通过事件绑定触发 Modal，支持使用 ref 引用组件。

Element 触发 Element 作为内容 自定义 DOM 无图标 表单数据返回 克隆元素
{{$data.elementModalLog.length > 0 ? $data.elementModalLog.join('\n') : '等待交互...'}}
// 通过 ref 获取 Element 组件触发 Modal
const btn = this.refs.modalTriggerBtn;

Modal.open({
    title: 'Element 触发',
    content: '通过 Solely Button 组件触发的对话框',
    onConfirm: () => {
        console.log('确定');
    }
});

// 使用 DOM 元素作为内容
const content = document.createElement('div');
content.innerHTML = '<solely-tag type="success">成功</solely-tag>';

Modal.open({
    title: '自定义内容',
    content: content,
    onConfirm: () => {}
});

// 完全自定义内容（无默认图标）
const customContent = document.createElement('div');
customContent.innerHTML = `
    <div style="display: flex; align-items: center; gap: 12px;">
        <span style="font-size: 32px;">✅</span>
        <div>
            <div style="font-weight: 500;">操作成功</div>
            <div style="color: var(--solely-text-secondary);">数据已保存</div>
        </div>
    </div>
`;

Modal.open({
    title: '自定义标题',
    content: customContent,
    showIcon: false,  // 去掉默认图标
    onConfirm: () => {}
});

// onConfirm 返回表单数据
const form = document.createElement('form');
form.innerHTML = '<input name="username" /><input name="email" />';

Modal.confirm({
    title: '填写信息',
    content: form,
    onConfirm: () => {
        const formData = new FormData(form);
        return {
            username: formData.get('username'),
            email: formData.get('email')
        };
    }
});

// 克隆元素（保留原始元素）
const content = document.createElement('div');
content.textContent = '内容';

Modal.open({
    title: '克隆元素',
    content: content,
    cloneElement: true,  // 克隆元素
    onConfirm: () => {}
});

API

Modal.open(options)

参数 说明 类型 默认值
title 对话框标题 string | HTMLElement -
content 对话框内容 string | HTMLElement -
type 对话框类型 'info' | 'success' | 'warning' | 'error' | 'confirm' 'info'
width 对话框宽度 number | string 520
mask 是否显示遮罩 boolean true
maskClosable 点击遮罩层是否关闭 boolean false
closable 是否显示关闭按钮 boolean true
showIcon 是否显示图标 boolean true
okText 确定按钮文字 string | HTMLElement '确定'
cancelText 取消按钮文字 string '取消'
okType 确定按钮类型 'primary' | 'default' | 'danger' 'primary'
showCancel 是否显示取消按钮 boolean type === 'confirm' ? true : false
buttons 自定义按钮数组（如果提供，将忽略 okText/cancelText/showCancel） ModalButton[] -
className 自定义类名 string -
style 自定义样式 Partial<CSSStyleDeclaration> -
cloneElement 是否克隆传入的 HTMLElement（title/content/okText/cancelText），默认不克隆以支持双向数据绑定 boolean false
onConfirm 确定按钮回调（支持异步）。返回 false 阻止关闭，返回其他值作为结果数据 () => void | false | any | Promise<void | false | any> -
onCancel 取消按钮回调 () => void -
onClose 关闭后回调 () => void -

ModalButton

参数 说明 类型 默认值
text 按钮文字 string -
type 按钮类型 'primary' | 'default' | 'danger' 'default'
style 按钮样式 Partial<CSSStyleDeclaration> -
onClick 点击回调（支持异步） () => void | Promise<void> -

快捷方法

方法 说明 参数 返回值
Modal.confirm 确认对话框 options: Omit<ModalOptions, 'type'> ModalInstance
Modal.info 信息对话框 options: Omit<ModalOptions, 'type'> ModalInstance
Modal.success 成功对话框 options: Omit<ModalOptions, 'type'> ModalInstance
Modal.warning 警告对话框 options: Omit<ModalOptions, 'type'> ModalInstance
Modal.error 错误对话框 options: Omit<ModalOptions, 'type'> ModalInstance
Modal.config 全局配置 options: Partial<ModalConfig> void
Modal.destroy 销毁所有对话框 - void

ModalInstance

方法 说明 参数
close() 关闭对话框 -
update(options) 更新对话框配置 options: Partial<ModalOptions>

ModalConfig

参数 说明 类型 默认值
width 默认宽度 number | string 520
mask 默认是否显示遮罩 boolean true
maskClosable 默认是否可点击遮罩关闭 boolean false
okText 默认确定按钮文字 string '确定'
cancelText 默认取消按钮文字 string '取消'
